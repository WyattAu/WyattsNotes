### Statistics

<PracticeProblem

  options={['7', '4', '6', '9']}

  explanation="The median is the middle value when data is arranged in order. With five values, the third value is the median: 3, 4, 7, 7, 9, so the median is 7. If there is an even number of values, the median is the average of the two middle values. The mean of this set is (3+4+7+7+9)/5 = 30/5 = 6, and the mode is 7 (most frequent value)."

<PracticeProblem

  options={['1/2', '1/10', '3/10', '7/10']}

  explanation="Total balls = 3 + 5 + 2 = 10. Probability of not blue = probability of red or green = (3 + 2) / 10 = 5/10 = 1/2. Alternatively, use the complement rule: P(not blue) = 1 - P(blue) = 1 - 5/10 = 5/10 = 1/2. The complement rule is useful when it is easier to calculate the probability of the event not happening."
