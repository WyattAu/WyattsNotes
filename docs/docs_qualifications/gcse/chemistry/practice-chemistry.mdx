### Quantitative Chemistry

<PracticeProblem
  question="What is the relative formula mass of CaCO3? (Relative atomic masses: Ca = 40, C = 12, O = 16)"
  options={['100', '116', '84', '68']}
  correctAnswer={0}
  explanation="The relative formula mass (M_r) is found by adding the relative atomic masses of all atoms in the formula. CaCO3 contains one Ca (40), one C (12), and three O (3 x 16 = 48). So M_r = 40 + 12 + 48 = 100."
