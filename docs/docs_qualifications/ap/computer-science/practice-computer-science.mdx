### Data Analysis and Networks

<PracticeProblem question="Which of the following describes the primary difference between lossy and lossless data compression?" options={[ 'Lossy compression permanently removes some data to achieve smaller file sizes, while lossless compression preserves all original data', 'Lossy compression is only used for text files, while lossless compression is used for images', 'Lossless compression always produces smaller files than lossy compression', 'Lossy compression cannot be reversed, while lossless compression can only be reversed once', ]} correctAnswer={0} explanation="Lossy compression achieves smaller file sizes by permanently discarding some data that is deemed less important or less perceptible to humans (such as certain frequencies in audio or color variations in images). The original data cannot be perfectly reconstructed. Lossless compression reduces file size without losing any data, allowing the original to be perfectly reconstructed. Lossy is typically used for media (JPEG, MP3) where minor quality loss is acceptable.
