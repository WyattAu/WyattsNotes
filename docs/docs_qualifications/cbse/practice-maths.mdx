<PracticeProblem
  question="Evaluate: $\\dfrac{d}{dx}\\left[\\tan^{-1}\\left(\\dfrac{x}{1+\\sqrt{1-x^2}}\\right)\\right]$ at $x = \\dfrac{1}{2}$:"

  correctAnswer={0}
  explanation="Let $x = \\sin\\theta$. Then the expression simplifies to $\\dfrac{d}{dx}\\left[\\dfrac{\\theta}{2}\\right] = \\dfrac{1}{2} \\cdot \\dfrac{1}{\\sqrt{1-x^2}}$. At $x = 1/2$, this equals $\\dfrac{1}{2} \\cdot \\dfrac{1}{\\sqrt{3}/2} = \\dfrac{1}{\\sqrt{3}}$. Alternatively, simplifying the composition yields the derivative $\\dfrac{1}{2\\sqrt{1-x^2}}$, and at $x = 1/2$: $\\dfrac{1}{2\\cdot\\sqrt{3}/2} = \\dfrac{1}{\\sqrt{3}}$."

<PracticeProblem

Since $\\int e^x(f(x) + f'(x))\\,dx = e^x f(x) + C$, we get $e^x \\tan(x/2) + C$."

<PracticeProblem
  question="If $f(x) = |\\cos x|$, then $f'(x)$ at $x = \\dfrac{\\pi}{2}$ is:"
  options={['$1$', '$-1$', 'Does not exist', '$0$']}
  correctAnswer={2}
  explanation="At $x = \\pi/2$, $\\cos x = 0$ and the function $|\\cos x|$ changes from decreasing (where $\\cos x > 0$, so $f'(x) = -\\sin x$ approaching $-1$ from the left) to increasing (where $\\cos x < 0$, so $f'(x) = \\sin x$ approaching $1$ from the right). The left-hand derivative is $-1$ and the right-hand derivative is $+1$, so the derivative does not exist at this cusp."
  difficulty="medium"
/>

<PracticeProblem
  question="If $P(A) = \\dfrac{3}{5}$, $P(B) = \\dfrac{1}{3}$, and $P(A \\cap B) = \\dfrac{1}{5}$, then $P(A \\cup B)$ is:"
  options={['$\\dfrac{13}{15}$', '$\\dfrac{11}{15}$', '$\\dfrac{4}{5}$', '$\\dfrac{2}{3}$']}
  correctAnswer={0}
  explanation="$P(A \\cup B) = P(A) + P(B) - P(A \\cap B) = \\dfrac{3}{5} + \\dfrac{1}{3} - \\dfrac{1}{5} = \\dfrac{2}{5} + \\dfrac{1}{3} = \\dfrac{6+5}{15} = \\dfrac{11}{15}$."
  difficulty="medium"
/>

<PracticeProblem
  question="A random variable $X$ has the probability distribution: $P(X = 0) = \\dfrac{1}{5}$, $P(X = 1) = \\dfrac{2}{5}$, $P(X = 2) = \\dfrac{2}{5}$. Find $E(X)$:"
  options={['$1.2$', '$1.0$', '$1.4$', '$1.6$']}
  correctAnswer={0}
  explanation="$E(X) = \\sum x_i P(X = x_i) = 0 \\times \\dfrac{1}{5} + 1 \\times \\dfrac{2}{5} + 2 \\times \\dfrac{2}{5} = 0 + \\dfrac{2}{5} + \\dfrac{4}{5} = \\dfrac{6}{5} = 1.2$."
  difficulty="medium"
/>

<PracticeProblem
  question="If $A$ and $B$ are independent events with $P(A) = 0.6$ and $P(B) = 0.4$, then $P(A \\cup B)$ is:"
  options={['$0.76$', '$0.24$', '$0.84$', '$0.64$']}
  correctAnswer={0}
  explanation="For independent events: $P(A \\cap B) = P(A) \\cdot P(B) = 0.6 \\times 0.4 = 0.24$. Then $P(A \\cup B) = P(A) + P(B) - P(A \\cap B) = 0.6 + 0.4 - 0.24 = 0.76$."
  difficulty="medium"
/>
