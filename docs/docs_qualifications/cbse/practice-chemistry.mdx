<PracticeProblem
  question="The activation energy of a reaction is 209.5 kJ mol$^{-1}$ and its pre-exponential factor $A = 1.0 \\times 10^{13}$ s$^{-1}$. The rate constant at 600 K (given $R = 8.314$ J K$^{-1}$ mol$^{-1}$) is approximately:"
  options={[

    '$2.15 \\times 10^{-6}$ s$^{-1}$',
    '$6.31 \\times 10^{-4}$ s$^{-1}$',
  ]}

  explanation="Using $k = Ae^{-E_a/RT}$: $E_a/RT = \\dfrac{209.5 \\times 10^3}{8.314 \\times 600} \\approx 41.97$. Since $e^{-41.97} \\approx 3.98 \\times 10^{-19}$: $k \\approx 3.98 \\times 10^{-6}$ s$^{-1}$. The closest option is $3.98 \\times 10^{-5}$ s$^{-1}$ (using the logarithmic approach: $\\log k = 13 - \\dfrac{209500}{2.303 \\times 8.314 \\times 600} \\approx -4.40$, so $k \\approx 3.98 \\times 10^{-5}$)."

## Solutions and P-Block Elements

<PracticeProblem
  question="The van't Hoff factor $i$ for 0.1 M $\\text{Al}_2(\\text{SO}_4)_3$ solution (assuming 100% dissociation) is:"
  options={['5', '2', '3', '6']}
  correctAnswer={0}
  explanation="$\\text{Al}_2(\\text{SO}_4)_3 \\to 2\\text{Al}^{3+} + 3\\text{SO}_4^{2-}$. Total ions produced $= 2 + 3 = 5$. The van't Hoff factor $i$ equals the total number of ions produced from one formula unit of the solute, so $i = 5$."
  difficulty="easy"
/>

## D-Block Elements and Polymers

<PracticeProblem
  question="Which of the following is a spin-only magnetic moment formula for a transition metal ion with $n$ unpaired electrons?"
  options={[
    '$\\mu = \\sqrt{n(n+2)}$ BM',
    '$\\mu = \\sqrt{4n(n+1)}$ BM',
    '$\\mu = n(n+2)$ BM',
    '$\\mu = \\sqrt{n+2}$ BM',
  ]}
  correctAnswer={0}
  explanation="The spin-only magnetic moment is given by $\\mu_s = \\sqrt{n(n+2)}$ Bohr magnetons (BM), where $n$ is the number of unpaired $d$ electrons. This formula derives from $\\mu = \\sqrt{4S(S+1)}$ BM where $S = n/2$, giving $\\mu = \\sqrt{4 \\cdot \\frac{n}{2} \\cdot \\left(\\frac{n}{2}+1\\right)} = \\sqrt{n(n+2)}$ BM."
  difficulty="medium"
/>
