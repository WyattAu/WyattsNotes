### Modulus-Argument Form

<PracticeProblem question="Express z = 1 + √3 i in modulus-argument (r, θ) form." options={['(2, π/3)', '(2, π/6)', '(√3, π/3)', '(2, 2π/3)']} correctAnswer={0} explanation="r = √(1² + (√3)²) = √(1 + 3) = 2. θ = arctan(√3/1) = π/3 (in the first quadrant). So z = 2(cos π/3 + i sin π/3)." difficulty="medium" />

<PracticeProblem question="What is |z₁z₂| if |z₁| = 3 and |z₂| = 5?" options={['8', '15', '2', '√34']} correctAnswer={1} explanation="The modulus of a product equals the product of the moduli: |z₁z₂| = |z₁| · |z₂| = 3 × 5 = 15." difficulty="easy" />

### De Moivre's Theorem

<PracticeProblem question="Using De Moivre's theorem, find (1 + i)⁶." options={['−8i', '8i', '8', '−8']} correctAnswer={0} explanation="r = √2, θ = π/4. By De Moivre: (√2 cis(π/4))⁶ = (√2)⁶ cis(6π/4) = 8 cis(3π/2) = 8(cos(3π/2) + i sin(3π/2)) = 8(0 + i(−1)) = −8i." difficulty="hard" />

<PracticeProblem question="Express cos(3θ) in terms of cos(θ) using De Moivre's theorem." options={['4cos³θ − 3cosθ', '3cosθ − 4cos³θ', 'cos³θ − 3cosθ', '3cos³θ − cosθ']} correctAnswer={0} explanation="By De Moivre: cos(3θ) + i·sin(3θ) = (cosθ + i·sinθ)³. Expanding real part: cos³θ − 3cosθ·sin²θ = cos³θ − 3cosθ(1 − cos²θ) = 4cos³θ − 3cosθ." difficulty="hard" />
