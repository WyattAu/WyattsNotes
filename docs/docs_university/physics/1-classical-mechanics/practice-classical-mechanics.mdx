### Energy

<PracticeProblem question="A particle moves in a potential V(x) = ½kx² − αx³ with k, α > 0. ### Oscillations

<PracticeProblem question="A damped harmonic oscillator has equation mẍ + bẋ + kx = 0 with m = 1 kg, k = 100 N/m. ### Central Forces

<PracticeProblem question="A satellite orbits Earth at radius r with speed v. ### Lagrangian Mechanics

<PracticeProblem question="A particle of mass m slides on a frictionless parabolic wire y = ½ax² under gravity. Using x as the generalised coordinate, what is the Lagrangian L?" options={['L = ½m(1 + a²x²)ẋ² − ½mgax²', 'L = ½mẋ² + ½mgax²', 'L = ½mẋ² − mgy', 'L = ½mv² + mgy']} correctAnswer={0} explanation="T = ½m(ẋ² + ẏ²) = ½m(ẋ² + a²x²ẋ²) = ½m(1 + a²x²)ẋ². <PracticeProblem question="For a simple pendulum with Lagrangian L = ½mL²θ̇² − mgL(1 − cosθ), what is the Euler-Lagrange equation?" options={[ 'mL²θ̈ + mgL sinθ = 0', 'mL²θ̈ − mgL sinθ = 0', 'mLθ̈ + mg sinθ = 0', 'mL²θ̈ + mgL cosθ = 0', ]} correctAnswer={0} explanation="Apply ∂L/∂θ − d/dt(∂L/∂θ̇) = 0. ∂L/∂θ = −mgL sinθ. ∂L/∂θ̇ = mL²θ̇. d/dt(∂L/∂θ̇) = mL²θ̈. So: −mgL sinθ − mL²θ̈ = 0 → mL²θ̈ + mgL sinθ = 0 → θ̈ + (g/L)sinθ = 0. For small angles: θ̈ + (g/L)θ = 0 (SHM with ω² = g/L)." difficulty="medium" />
