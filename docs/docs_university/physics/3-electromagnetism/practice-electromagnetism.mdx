### Circuits and Electromagnetic Waves

  explanation="Resonant frequency: f_0 = 1 / (2*pi*sqrt(LC)) = 1 / (2*pi*sqrt(50 x 10^-3 * 20 x 10^-6)) = 1 / (2*pi*sqrt(1.0 x 10^-6)) = 1 / (2*pi*1.0 x 10^-3) = 159 Hz. At resonance, the impedance is purely resistive (Z = R = 100 Ohm), the current is maximum, and the reactances cancel: X_L = X_C."
