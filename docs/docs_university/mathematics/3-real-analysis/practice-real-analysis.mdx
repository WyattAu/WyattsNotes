### Integration

<PracticeProblem
  question="Which of the following functions is Riemann integrable on $[0, 1]$?"
  options={[
    'The Dirichlet function (1 on rationals, 0 on irrationals) is NOT Riemann integrable',
    'Every bounded function on $[0, 1]$ is Riemann integrable',
    'Only continuous functions are Riemann integrable',
    'A function with countably many discontinuities may fail to be Riemann integrable',
  ]}
  correctAnswer={0}
  explanation="The Dirichlet function is the classic example of a bounded function that is NOT Riemann integrable: its lower sums are always 0 and upper sums are always 1, so the lower and upper integrals disagree. A bounded function on $[a, b]$ is Riemann integrable if and only if its set of discontinuities has measure zero (Lebesgue's criterion). Functions with countably many discontinuities ARE Riemann integrable since countable sets have measure zero."
  difficulty="medium"
