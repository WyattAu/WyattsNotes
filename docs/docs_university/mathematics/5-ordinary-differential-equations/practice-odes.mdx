<PracticeProblem
  question="Using undetermined coefficients, what is a particular solution of $y'' + y = 3\cos(x)$?"
  options={[

  explanation="The complementary solution is $y_c = C_1 \cos(x) + C_2 \sin(x)$. Since $\cos(x)$ appears in $y_c$, the standard guess $A\cos(x) + B\sin(x)$ would fail. Substituting: $y_p'' + y_p = -2A\sin(x) + 2B\cos(x) = 3\cos(x)$. So $A = 0, B = 3/2$, giving $y_p = \frac{3}{2}x\sin(x)$."

<PracticeProblem

    '$a_{n+1} = \frac{a_n}{n+1}$ for $n \geq 0$',

  explanation="Let $y = \sum_{n=0}^{\infty} a_n x^n$. Then $y'' = \sum_{n=0}^{\infty} (n+2)(n+1) a_{n+2} x^n$ and $xy = \sum_{n=1}^{\infty} a_{n-1} x^n$. Substituting into $y'' - xy = 0$: for $n = 0$, $2a_2 = 0$; for $n \geq 1$: $(n+2)(n+1)a_{n+2} - a_{n-1} = 0$, giving $a_{n+2} = a_{n-1}/((n+2)(n+1))$. This is Airy's equation."
