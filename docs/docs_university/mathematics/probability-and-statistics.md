## 1. Probability Spaces

### 1.1 Sample Spaces and Events

A **probability space** is a triple $(\Omega, \mathcal◆LB◆F◆RB◆, P)$ where:

- $\Omega$ is the **sample space** (set of all possible outcomes).
- $\mathcal◆LB◆F◆RB◆$ is a **sigma-algebra** (collection of events) on $\Omega$.
- $P : \mathcal◆LB◆F◆RB◆ \to [0,1]$ is a **probability measure**.

### 1.2 Sigma-Algebras

**Definition.** A **sigma-algebra** (or $\sigma$-algebra) $\mathcal◆LB◆F◆RB◆$ on a set $\Omega$ is a
collection of subsets Of $\Omega$ satisfying:

1. $\Omega \in \mathcal◆LB◆F◆RB◆$.
2. If $A \in \mathcal◆LB◆F◆RB◆$Then $A^c \in \mathcal◆LB◆F◆RB◆$ (closed under complementation).
3. If $A_1, A_2, A_3, \ldots \in \mathcal◆LB◆F◆RB◆$Then $\bigcup_◆LB◆i=1◆RB◆^◆LB◆\infty◆RB◆ A_i \in \mathcal◆LB◆F◆RB◆$
   (closed under countable unions).

_Remark._ From (2) and (3), $\mathcal◆LB◆F◆RB◆$ is also closed under countable intersections (by De
Morgan's laws). The pair $(\Omega, \mathcal◆LB◆F◆RB◆)$ is called a **measurable space**.

**Example 1.1.** For any set $\Omega$The **trivial sigma-algebra** is `\\{\\emptyset, \\Omega\\}`
and the **power Set** $\mathcal◆LB◆P◆RB◆(\Omega)$ is also a sigma-algebra.

**Example 1.2.** If $\Omega = \{1, 2, 3, 4, 5, 6\}$ (a fair die), then
$\mathcal◆LB◆F◆RB◆ = \mathcal◆LB◆P◆RB◆(\Omega)$ contains all $2^6 = 64$ subsets. This is the sigma-algebra used
for finite Sample spaces.

**Example 1.3.** For $\Omega = \mathbb◆LB◆R◆RB◆$The **Borel sigma-algebra** $\mathcal◆LB◆B◆RB◆$ is the smallest
sigma-algebra Containing all open intervals. It is generated by taking countable unions,
intersections, and complements of open Sets. We write $(\mathbb◆LB◆R◆RB◆, \mathcal◆LB◆B◆RB◆)$.

**Proposition 1.0.** The intersection of any collection of sigma-algebras on $\Omega$ is a
sigma-algebra.

_Proof._ Let `\\{\\mathcal{{'}F{}'}_\\alpha\\}` be a collection of sigma-algebras. Then: (1)
$\Omega \in \mathcal◆LB◆F◆RB◆_\alpha$ for all $\alpha$So $\Omega \in \bigcap_\alpha \mathcal◆LB◆F◆RB◆_\alpha$.
(2) If $A \in \bigcap_\alpha \mathcal◆LB◆F◆RB◆_\alpha$Then $A \in \mathcal◆LB◆F◆RB◆_\alpha$ for all $\alpha$So
$A^c \in \mathcal◆LB◆F◆RB◆_\alpha$ for all $\alpha$Hence $A^c \in \bigcap_\alpha \mathcal◆LB◆F◆RB◆_\alpha$. (3)
Countable unions follow similarly. $\blacksquare$

_Intuition._ This proposition guarantees that for any collection of subsets $\mathcal◆LB◆E◆RB◆$There
exists a **smallest** Sigma-algebra containing $\mathcal◆LB◆E◆RB◆$Called the sigma-algebra **generated
by** $\mathcal◆LB◆E◆RB◆$Denoted $\sigma(\mathcal◆LB◆E◆RB◆)$.

### 1.3 Axioms of Probability (Kolmogorov)

1. **Non-negativity**: $P(A) \geq 0$ for all $A \in \mathcal◆LB◆F◆RB◆$.
2. **Normalisation**: $P(\Omega) = 1$.
3. **Countable additivity**: If $A_1, A_2, \ldots$ are pairwise disjoint events, then

$$P\left(\bigcup_◆LB◆i=1◆RB◆^◆LB◆\infty◆RB◆ A_i\right) = \sum_◆LB◆i=1◆RB◆^◆LB◆\infty◆RB◆ P(A_i)$$

### 1.4 Basic Properties

**Proposition 1.1.** $P(\emptyset) = 0$.

_Proof._ $\Omega = \Omega \cup \emptyset$ (disjoint union), so
$P(\Omega) = P(\Omega) + P(\emptyset)$ Hence $P(\emptyset) = 0$. $\blacksquare$

**Proposition 1.2 (Complement).** $P(A^c) = 1 - P(A)$.

**Proposition 1.3 (Monotonicity).** If $A \subseteq B$Then $P(A) \leq P(B)$.

_Proof._ Write $B = A \cup (B \setminus A)$A disjoint union. By countable additivity,
$P(B) = P(A) + P(B \setminus A) \geq P(A)$ since $P(B \setminus A) \geq 0$. $\blacksquare$

**Proposition 1.4 (Inclusion-Exclusion).** For any two events $A, B$:

$$P(A \cup B) = P(A) + P(B) - P(A \cap B)$$

_Proof._ Write $A \cup B = A \cup (B \setminus A)$ as a disjoint union. Then
$P(A \cup B) = P(A) + P(B \setminus A)$. Since $B = (B \setminus A) \cup (A \cap B)$ is also
disjoint, $P(B) = P(B \setminus A) + P(A \cap B)$So $P(B \setminus A) = P(B) - P(A \cap B)$.
Substituting gives the Result. $\blacksquare$

**Proposition 1.5 (General Inclusion-Exclusion).** For events $A_1, \ldots, A_n$:

$$P\left(\bigcup_◆LB◆i=1◆RB◆^n A_i\right) = \sum_◆LB◆i◆RB◆ P(A_i) - \sum_◆LB◆i \lt j◆RB◆ P(A_i \cap A_j) + \sum_◆LB◆i \lt j \lt k◆RB◆ P(A_i \cap A_j \cap A_k) - \cdots + (-1)^◆LB◆n+1◆RB◆ P(A_1 \cap \cdots \cap A_n)$$

**Proposition 1.6 (Boole's Inequality).** $P(A \cup B) \leq P(A) + P(B)$. More generally,

$$P\left(\bigcup_◆LB◆i=1◆RB◆^n A_i\right) \leq \sum_◆LB◆i=1◆RB◆^n P(A_i)$$

### 1.5 Conditional Probability

**Definition.** The **conditional probability** of $A$ given $B$ (where $P(B) \gt 0$) is

$$P(A \mid B) = \frac◆LB◆P(A \cap B)◆RB◆◆LB◆P(B)◆RB◆$$

_Intuition._ Conditioning on $B$ restricts the sample space to $B$ and rescales so that
$P(B \mid B) = 1$.

**Theorem 1.1 (Law of Total Probability).** If $B_1, \ldots, B_n$ partition $\Omega$ with
$P(B_i) \gt 0$:

$$P(A) = \sum_◆LB◆i=1◆RB◆^n P(A \mid B_i) P(B_i)$$

_Proof._ Since $B_1, \ldots, B_n$ partition $\Omega$We have $A = \bigcup_◆LB◆i=1◆RB◆^n (A \cap B_i)$
(disjoint union). By countable additivity:

$$P(A) = \sum_◆LB◆i=1◆RB◆^n P(A \cap B_i) = \sum_◆LB◆i=1◆RB◆^n P(A \mid B_i)\, P(B_i)$$

$\blacksquare$

**Theorem 1.2 (Bayes' Theorem).** For events $A$ and $B$ with $P(B) \gt 0$:

$$P(A \mid B) = \frac◆LB◆P(B \mid A) P(A)◆RB◆◆LB◆P(B)◆RB◆$$

In the partition form:

$$P(B_j \mid A) = \frac◆LB◆P(A \mid B_j) P(B_j)◆RB◆◆LB◆\sum_◆LB◆i=1◆RB◆^n P(A \mid B_i) P(B_i)◆RB◆$$

_Proof._ By definition, $P(A \mid B) = P(A \cap B)/P(B)$ and $P(B \mid A) = P(A \cap B)/P(A)$.
Solving the Second for $P(A \cap B) = P(B \mid A) P(A)$ and substituting into the first gives Bayes'
theorem. The partition Form follows by applying the law of total probability to the denominator
$P(B)$. $\blacksquare$

### 1.6 Worked Examples

**Problem 1.1.** A bag contains 4 red and 6 blue marbles. Two marbles are drawn without replacement.
What is the Probability that both are red?

<details>
<summary>Solution</summary>

Let $R_1$ be the event "first marble is red" and $R_2$ be "second marble is red." Then:

$$P(R_1 \cap R_2) = P(R_1)\, P(R_2 \mid R_1) = \frac◆LB◆4◆RB◆◆LB◆10◆RB◆ \cdot \frac◆LB◆3◆RB◆◆LB◆9◆RB◆ = \frac◆LB◆12◆RB◆◆LB◆90◆RB◆ = \frac◆LB◆2◆RB◆◆LB◆15◆RB◆$$

</details>

**Problem 1.2.** A disease affects 1% of a population. A test has sensitivity 95%
($P(\mathrm◆LB◆positive◆RB◆ \mid \mathrm◆LB◆disease◆RB◆) = 0.95$) And specificity 90%
($P(\mathrm◆LB◆negative◆RB◆ \mid \mathrm◆LB◆healthy◆RB◆) = 0.90$). If a person tests positive, what is the
Probability they have the disease?

<details>
<summary>Solution</summary>

Let $D$ = "has disease" and $+$ = "tests positive." We want $P(D \mid +)$.

By Bayes' theorem:

$$P(D \mid +) = \frac◆LB◆P(+ \mid D)\, P(D)◆RB◆◆LB◆P(+ \mid D)\, P(D) + P(+ \mid D^c)\, P(D^c)◆RB◆$$

$$= \frac◆LB◆0.95 \times 0.01◆RB◆◆LB◆0.95 \times 0.01 + 0.10 \times 0.99◆RB◆ = \frac◆LB◆0.0095◆RB◆◆LB◆0.0095 + 0.099◆RB◆ = \frac◆LB◆0.0095◆RB◆◆LB◆0.1085◆RB◆ \approx 0.0876$$

So even with a positive test, there is only about an 8.8% chance of having the disease, due to the
low prior Probability (base rate fallacy). $\blacksquare$

</details>

**Proposition 1.7 (General Inclusion-Exclusion).** For events $A_1, \ldots, A_n$:

$$P\left(\bigcup_◆LB◆i=1◆RB◆^n A_i\right) = \sum_◆LB◆k=1◆RB◆^n (-1)^◆LB◆k+1◆RB◆ \sum_◆LB◆1 \leq i_1 \lt \cdots \lt i_k \leq n◆RB◆ P(A_◆LB◆i_1◆RB◆ \cap \cdots \cap A_◆LB◆i_k◆RB◆)$$

_Proof._ By induction on $n$. The base case $n = 1$ is trivial. Assume the result holds for $n$
events. For $n + 1$ Events:

$$P\left(\bigcup_◆LB◆i=1◆RB◆^◆LB◆n+1◆RB◆ A_i\right) = P\left(\bigcup_◆LB◆i=1◆RB◆^n A_i\right) + P(A_◆LB◆n+1◆RB◆) - P\left(\bigcup_◆LB◆i=1◆RB◆^n A_i \cap A_◆LB◆n+1◆RB◆\right)$$

Applying the induction hypothesis to both union terms and rearranging yields the formula for
$n + 1$. $\blacksquare$

**Problem 1.3.** Three machines A, B, C produce items. Machine A produces 50% of items with 2%
defective, B produces 30% with 1% defective, and C produces 20% with 3% defective. An item is found
to be defective. What is the Probability it was produced by machine A?

<details>
<summary>Solution</summary>

Let $D$ = "defective" and $A$, $B$, $C$ denote production by each machine.

$P(A) = 0.5$$P(B) = 0.3$$P(C) = 0.2$. $P(D \mid A) = 0.02$$P(D \mid B) = 0.01$$P(D \mid C) = 0.03$.

$$P(A \mid D) = \frac◆LB◆P(D \mid A)\, P(A)◆RB◆◆LB◆P(D \mid A)\, P(A) + P(D \mid B)\, P(B) + P(D \mid C)\, P(C)◆RB◆$$

$$= \frac◆LB◆0.02 \times 0.5◆RB◆◆LB◆0.02 \times 0.5 + 0.01 \times 0.3 + 0.03 \times 0.2◆RB◆ = \frac◆LB◆0.010◆RB◆◆LB◆0.010 + 0.003 + 0.006◆RB◆ = \frac◆LB◆0.010◆RB◆◆LB◆0.019◆RB◆ \approx 0.526$$

</details>

:::caution Common Pitfall People often confuse $P(A \mid B)$ with $P(B \mid A)$. In medical testing,
$P(\mathrm◆LB◆disease◆RB◆ \mid \mathrm◆LB◆positive◆RB◆)$ Is much lower than
$P(\mathrm◆LB◆positive◆RB◆ \mid \mathrm◆LB◆disease◆RB◆)$ due to low base rates. Always apply Bayes' Theorem
rigorously. :::

## 2. Random Variables

### 2.1 Definition

A **random variable** is a measurable function $X : \Omega \to \mathbb◆LB◆R◆RB◆$. Measurability means that
for every Borel Set
$B \subseteq \mathbb◆LB◆R◆RB◆$, $\{X^◆LB◆-1◆RB◆(B) : \omega \in \Omega, X(\omega) \in B\} \in \mathcal◆LB◆F◆RB◆$.

- **Discrete random variable**: takes values in a countable set.
- **Continuous random variable**: has a probability density function (PDF).

**Example 2.1 (Discrete).** Roll a fair die. Define $X(\omega) = \omega$. Then $X$ takes values in
$\{1, 2, 3, 4, 5, 6\}$ With $P(X = k) = 1/6$ for each $k$.

**Example 2.2 (Discrete — Indicator).** For any event $A$The **indicator random variable**
$\mathbf◆LB◆1◆RB◆_A$ equals 1 if $A$ occurs and 0 otherwise. Then $E[\mathbf◆LB◆1◆RB◆_A] = P(A)$ and
$\mathrm◆LB◆Var◆RB◆(\mathbf◆LB◆1◆RB◆_A) = P(A)(1 - P(A))$.

**Example 2.3 (Continuous).** Let $X \sim \mathrm◆LB◆Uniform◆RB◆(0, 1)$. Then $X$ is the identity on
$(0, 1)$ with PDF $f(x) = 1$ for $x \in (0, 1)$ and $f(x) = 0$ otherwise.

**Example 2.4 (Mixed).** A random variable can be neither purely discrete nor purely continuous. For
instance, if $X = 0$ with probability $1/2$ and $X \sim \mathrm◆LB◆Exp◆RB◆(1)$ with probability $1/2$Then
$X$ has an atom at 0 And a continuous part on $(0, \infty)$.

### 2.2 Cumulative Distribution Function

The **CDF** of a random variable $X$ is

$$F_X(x) = P(X \leq x)$$

**Theorem 2.1 (Properties of the CDF).**

1. $\lim_◆LB◆x \to -\infty◆RB◆ F(x) = 0$, $\lim_◆LB◆x \to +\infty◆RB◆ F(x) = 1$.
2. $F$ is non-decreasing: if $a \lt b$Then $F(a) \leq F(b)$.
3. $F$ is right-continuous: $\lim_◆LB◆x \to a^+◆RB◆ F(x) = F(a)$.
4. $P(a \lt X \leq b) = F(b) - F(a)$.
5. $P(X = a) = F(a) - F(a^-)$ where $F(a^-) = \lim_◆LB◆x \to a^-◆RB◆ F(x)$.
6. $F$ has at most countably many points of discontinuity.

_Proof._ (1) By monotonicity of $P$: $F(x) = P(X \leq x) \leq P(\Omega) = 1$ and as $x \to \infty$
$\{X \leq x\} \uparrow \Omega$So by continuity from below, $F(x) \to 1$. Similarly as
$x \to -\infty$ $\{X \leq x\} \downarrow \emptyset$ and $F(x) \to 0$.

(2) If $a \lt b$Then $\{X \leq a\} \subseteq \{X \leq b\}$So by monotonicity of $P$
$F(a) \leq F(b)$.

(3) Let $x_n \downarrow a$. Then $\{X \leq x_n\} \downarrow \{X \leq a\}$ (since
$\{X \leq a\} = \bigcap_n \{X \leq x_n\}$). By continuity from above of $P$, $F(x_n) \to F(a)$.

(4) $P(a \lt X \leq b) = P(X \leq b) - P(X \leq a) = F(b) - F(a)$.

(5) $P(X = a) = P(X \leq a) - P(X \lt a) = F(a) - \lim_◆LB◆x \uparrow a◆RB◆ F(x) = F(a) - F(a^-)$.

(6) Since $F$ is non-decreasing, it can have at most countably many jump discontinuities (the sum of
all jumps must Be bounded by 1). $\blacksquare$

### 2.3 Probability Mass Function (Discrete)

For a discrete random variable $X$ with values $\{x_1, x_2, \ldots\}$:

$$f_X(x) = P(X = x) = \begin◆LB◆cases◆RB◆ p_i & \mathrm◆LB◆if◆RB◆  x = x_i \\ 0 & \mathrm◆LB◆otherwise◆RB◆ \end◆LB◆cases◆RB◆$$

Where $p_i \geq 0$ and $\sum_i p_i = 1$.

### 2.4 Probability Density Function (Continuous)

A random variable $X$ is continuous if there exists a function $f_X \geq 0$ such that

$$P(a \leq X \leq b) = \int_a^b f_X(x)\, dx$$

And $\int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ f_X(x)\, dx = 1$.

Note: $f_X(x)$ is not a probability; it is a probability density. For continuous $X$, $P(X = x) = 0$
For any individual $x$.

### 2.5 Functions of Random Variables

**Proposition 2.1.** If $X$ is a continuous random variable with PDF $f_X$ and $g$ is a strictly
monotone Differentiable function, then $Y = g(X)$ has PDF

$$f_Y(y) = f_X(g^◆LB◆-1◆RB◆(y)) \cdot \left|\frac◆LB◆d◆RB◆◆LB◆dy◆RB◆ g^◆LB◆-1◆RB◆(y)\right|$$

_Proof._ Suppose $g$ is strictly increasing. Then
$F_Y(y) = P(Y \leq y) = P(X \leq g^◆LB◆-1◆RB◆(y)) = F_X(g^◆LB◆-1◆RB◆(y))$. Differentiating:
$f_Y(y) = f_X(g^◆LB◆-1◆RB◆(y)) \cdot (g^◆LB◆-1◆RB◆)'(y)$. For decreasing $g$The inequality reverses, Introducing
a minus sign. Both cases are captured by the absolute value. $\blacksquare$

**Problem 2.1.** Let $X \sim \mathrm◆LB◆Uniform◆RB◆(0, 1)$. Find the distribution of $Y = -\ln X$.

<details>
<summary>Solution</summary>

Here $g(x) = -\ln x$Which is strictly decreasing on $(0, 1)$. The inverse is $g^◆LB◆-1◆RB◆(y) = e^◆LB◆-y◆RB◆$
for $y \gt 0$. We have $(g^◆LB◆-1◆RB◆)'(y) = -e^◆LB◆-y◆RB◆$.

$$f_Y(y) = f_X(e^◆LB◆-y◆RB◆) \cdot |-e^◆LB◆-y◆RB◆| = 1 \cdot e^◆LB◆-y◆RB◆ = e^◆LB◆-y◆RB◆, \quad y \gt 0$$

This is the $\mathrm◆LB◆Exp◆RB◆(1)$ distribution. $\blacksquare$

</details>

**Problem 2.2.** Let $X \sim N(0, 1)$. Find the distribution of $Y = X^2$.

<details>
<summary>Solution</summary>

The function $g(x) = x^2$ is not monotone, so we must split into cases.

For $y \gt 0$:

$$F_Y(y) = P(X^2 \leq y) = P(-\sqrt◆LB◆y◆RB◆ \leq X \leq \sqrt◆LB◆y◆RB◆) = \Phi(\sqrt◆LB◆y◆RB◆) - \Phi(-\sqrt◆LB◆y◆RB◆) = 2\Phi(\sqrt◆LB◆y◆RB◆) - 1$$

Differentiating:

$$f_Y(y) = 2 \cdot \phi(\sqrt◆LB◆y◆RB◆) \cdot \frac◆LB◆1◆RB◆◆LB◆2\sqrt◆LB◆y◆RB◆◆RB◆ = \frac◆LB◆1◆RB◆◆LB◆\sqrt◆LB◆y◆RB◆◆RB◆ \cdot \frac◆LB◆1◆RB◆◆LB◆\sqrt◆LB◆2\pi◆RB◆◆RB◆ e^◆LB◆-y/2◆RB◆ = \frac◆LB◆1◆RB◆◆LB◆\sqrt◆LB◆2\pi y◆RB◆◆RB◆ e^◆LB◆-y/2◆RB◆$$

This is the PDF of a $\chi^2_1$ (chi-squared with 1 degree of freedom) distribution, which equals
$\mathrm◆LB◆Gamma◆RB◆(1/2, 1/2)$. $\blacksquare$

</details>

### 2.6 Quantile Function

**Definition.** The **quantile function** (or inverse CDF) of a random variable $X$ with CDF $F$ is

$$F^◆LB◆-1◆RB◆(p) = \inf\{x : F(x) \geq p\}, \quad 0 \lt p \lt 1$$

_Remark._ If $F$ is strictly increasing, then $F^◆LB◆-1◆RB◆(p)$ is the unique $x$ such that $F(x) = p$.
For discrete Distributions, $F^◆LB◆-1◆RB◆$ is the generalised inverse.

**Theorem 2.2 (Probability Integral Transform).** If $X$ has a continuous CDF $F$Then
$F(X) \sim \mathrm◆LB◆Uniform◆RB◆(0, 1)$.

_Proof._ For $u \in (0, 1)$: $P(F(X) \leq u) = P(X \leq F^◆LB◆-1◆RB◆(u)) = F(F^◆LB◆-1◆RB◆(u)) = u$.
$\blacksquare$

_Intuition._ This theorem is the foundation of **inverse transform sampling**: to generate from any
distribution With CDF $F$Draw $U \sim \mathrm◆LB◆Uniform◆RB◆(0, 1)$ and compute $X = F^◆LB◆-1◆RB◆(U)$.

### 2.7 Order Statistics

**Definition.** Let $X_1, \ldots, X_n$ be i.i.d. With CDF $F$ and PDF $f$. The **order statistics**
$X_◆LB◆(1)◆RB◆ \leq X_◆LB◆(2)◆RB◆ \leq \cdots \leq X_◆LB◆(n)◆RB◆$ are the sorted values.

**Theorem 2.3.** The PDF of the $k$-th order statistic $X_◆LB◆(k)◆RB◆$ is

$$f_◆LB◆X_◆LB◆(k)◆RB◆◆RB◆(x) = \frac◆LB◆n!◆RB◆◆LB◆(k-1)!(n-k)!◆RB◆\, [F(x)]^◆LB◆k-1◆RB◆\, [1 - F(x)]^◆LB◆n-k◆RB◆\, f(x)$$

_Proof._ For $X_◆LB◆(k)◆RB◆ \leq x$ to hold, at least $k$ of the $X_i$ must be $\leq x$. The event
$X_◆LB◆(k)◆RB◆ \in (x, x + dx)$ Requires exactly $k - 1$ observations below $x$One in $(x, x + dx)$And
$n - k$ above $x + dx$:

$$f_◆LB◆X_◆LB◆(k)◆RB◆◆RB◆(x)\, dx = \binom◆LB◆n◆RB◆◆LB◆k-1, 1, n-k◆RB◆\, [F(x)]^◆LB◆k-1◆RB◆\, f(x)\, dx\, [1 - F(x)]^◆LB◆n-k◆RB◆$$

Which gives the result after cancelling $dx$. $\blacksquare$

**Problem 2.3.** Let $X_1, X_2, X_3$ be i.i.d. $\mathrm◆LB◆Uniform◆RB◆(0, 1)$. Find the PDF of the median
$X_◆LB◆(2)◆RB◆$.

<details>
<summary>Solution</summary>

Here $n = 3$, $k = 2$$F(x) = x$$f(x) = 1$ on $(0, 1)$.

$$f_◆LB◆X_◆LB◆(2)◆RB◆◆RB◆(x) = \frac◆LB◆3!◆RB◆◆LB◆1! \cdot 1!◆RB◆\, x^◆LB◆1◆RB◆\, (1 - x)^◆LB◆1◆RB◆ \cdot 1 = 6x(1 - x), \quad 0 \lt x \lt 1$$

This is a $\mathrm◆LB◆Beta◆RB◆(2, 2)$ distribution. $\blacksquare$

</details>

## 3. Common Distributions

### 3.1 Discrete Distributions

**Bernoulli.** $X \sim \mathrm◆LB◆Bernoulli◆RB◆(p)$: $P(X = 1) = p$, $P(X = 0) = 1 - p$.

$$E[X] = p, \quad \mathrm◆LB◆Var◆RB◆(X) = p(1 - p)$$

$$M_X(t) = 1 - p + pe^t$$

_Proof of MGF:_ $M_X(t) = E[e^◆LB◆tX◆RB◆] = e^◆LB◆t \cdot 0◆RB◆(1 - p) + e^◆LB◆t \cdot 1◆RB◆ \cdot p = 1 - p + pe^t$.
$\blacksquare$

**Binomial.** $X \sim \mathrm◆LB◆Bin◆RB◆(n, p)$: number of successes in $n$ independent Bernoulli trials.

$$P(X = k) = \binom◆LB◆n◆RB◆◆LB◆k◆RB◆ p^k (1-p)^◆LB◆n-k◆RB◆, \quad k = 0, 1, \ldots, n$$

$$E[X] = np, \quad \mathrm◆LB◆Var◆RB◆(X) = np(1-p)$$

$$M_X(t) = (1 - p + pe^t)^n$$

_Proof of MGF:_ Since $X = \sum_◆LB◆i=1◆RB◆^n X_i$ where $X_i \sim \mathrm◆LB◆Bernoulli◆RB◆(p)$ are independent:

$$M_X(t) = \prod_◆LB◆i=1◆RB◆^n M_◆LB◆X_i◆RB◆(t) = (1 - p + pe^t)^n \quad \blacksquare$$

**Geometric.** $X \sim \mathrm◆LB◆Geom◆RB◆(p)$: number of trials until the first success (counting the
success).

$$P(X = k) = (1 - p)^◆LB◆k-1◆RB◆ p, \quad k = 1, 2, 3, \ldots$$

$$E[X] = \frac◆LB◆1◆RB◆◆LB◆p◆RB◆, \quad \mathrm◆LB◆Var◆RB◆(X) = \frac◆LB◆1 - p◆RB◆◆LB◆p^2◆RB◆$$

$$M_X(t) = \frac◆LB◆pe^t◆RB◆◆LB◆1 - (1 - p)e^t◆RB◆, \quad \mathrm◆LB◆for◆RB◆  t \lt -\ln(1 - p)$$

_Proof that $E[X] = 1/p$:_

$$E[X] = \sum_◆LB◆k=1◆RB◆^◆LB◆\infty◆RB◆ k(1-p)^◆LB◆k-1◆RB◆p = p \sum_◆LB◆k=1◆RB◆^◆LB◆\infty◆RB◆ k(1-p)^◆LB◆k-1◆RB◆ = p \cdot \frac◆LB◆1◆RB◆◆LB◆(1 - (1-p))^2◆RB◆ = \frac◆LB◆p◆RB◆◆LB◆p^2◆RB◆ = \frac◆LB◆1◆RB◆◆LB◆p◆RB◆$$

Using the identity $\sum_◆LB◆k=1◆RB◆^◆LB◆\infty◆RB◆ kr^◆LB◆k-1◆RB◆ = 1/(1-r)^2$ for $|r| \lt 1$. $\blacksquare$

**Negative Binomial.** $X \sim \mathrm◆LB◆NegBin◆RB◆(r, p)$: number of trials until the $r$-th success.

$$P(X = k) = \binom◆LB◆k-1◆RB◆◆LB◆r-1◆RB◆ p^r (1-p)^◆LB◆k-r◆RB◆, \quad k = r, r+1, \ldots$$

$$E[X] = \frac◆LB◆r◆RB◆◆LB◆p◆RB◆, \quad \mathrm◆LB◆Var◆RB◆(X) = \frac◆LB◆r(1-p)◆RB◆◆LB◆p^2◆RB◆$$

**Hypergeometric.** $X \sim \mathrm◆LB◆Hypergeometric◆RB◆(N, K, n)$: sampling $n$ items without replacement
from a population Of $N$ containing $K$ "successes."

$$P(X = k) = \frac◆LB◆\binom◆LB◆K◆RB◆◆LB◆k◆RB◆\binom◆LB◆N-K◆RB◆◆LB◆n-k◆RB◆◆RB◆◆LB◆\binom◆LB◆N◆RB◆◆LB◆n◆RB◆◆RB◆, \quad k = \max(0, n + K - N), \ldots, \min(n, K)$$

$$E[X] = n \cdot \frac◆LB◆K◆RB◆◆LB◆N◆RB◆, \quad \mathrm◆LB◆Var◆RB◆(X) = n \cdot \frac◆LB◆K◆RB◆◆LB◆N◆RB◆ \cdot \frac◆LB◆N-K◆RB◆◆LB◆N◆RB◆ \cdot \frac◆LB◆N-n◆RB◆◆LB◆N-1◆RB◆$$

_Remark._ The factor $(N - n)/(N - 1)$ is the **finite population correction**. When $n \ll N$The
Hypergeometric is well-approximated by $\mathrm◆LB◆Bin◆RB◆(n, K/N)$.

**Poisson.** $X \sim \mathrm◆LB◆Poisson◆RB◆(\lambda)$: models rare events occurring at rate $\lambda$.

$$P(X = k) = \frac◆LB◆e^◆LB◆-\lambda◆RB◆ \lambda^k◆RB◆◆LB◆k!◆RB◆, \quad k = 0, 1, 2, \ldots$$

$$E[X] = \lambda, \quad \mathrm◆LB◆Var◆RB◆(X) = \lambda$$

$$M_X(t) = \exp\left(\lambda(e^t - 1)\right)$$

_Proof of MGF:_

$$M_X(t) = \sum_◆LB◆k=0◆RB◆^◆LB◆\infty◆RB◆ e^◆LB◆tk◆RB◆ \frac◆LB◆e^◆LB◆-\lambda◆RB◆ \lambda^k◆RB◆◆LB◆k!◆RB◆ = e^◆LB◆-\lambda◆RB◆ \sum_◆LB◆k=0◆RB◆^◆LB◆\infty◆RB◆ \frac◆LB◆(\lambda e^t)^k◆RB◆◆LB◆k!◆RB◆ = e^◆LB◆-\lambda◆RB◆ \cdot e^◆LB◆\lambda e^t◆RB◆ = \exp(\lambda(e^t - 1))$$

$\blacksquare$

_Proof that $E[X] = \lambda$:_

$$E[X] = \sum_◆LB◆k=0◆RB◆^◆LB◆\infty◆RB◆ k \frac◆LB◆e^◆LB◆-\lambda◆RB◆ \lambda^k◆RB◆◆LB◆k!◆RB◆ = e^◆LB◆-\lambda◆RB◆ \sum_◆LB◆k=1◆RB◆^◆LB◆\infty◆RB◆ \frac◆LB◆\lambda^k◆RB◆◆LB◆(k-1)!◆RB◆ = e^◆LB◆-\lambda◆RB◆ \lambda \sum_◆LB◆k=0◆RB◆^◆LB◆\infty◆RB◆ \frac◆LB◆\lambda^k◆RB◆◆LB◆k!◆RB◆ = \lambda$$

$\blacksquare$

### 3.2 Continuous Distributions

**Uniform.** $X \sim \mathrm◆LB◆Uniform◆RB◆(a, b)$:

$$f(x) = \begin◆LB◆cases◆RB◆ \frac◆LB◆1◆RB◆◆LB◆b - a◆RB◆ & \mathrm◆LB◆if◆RB◆  a \leq x \leq b \\ 0 & \mathrm◆LB◆otherwise◆RB◆ \end◆LB◆cases◆RB◆$$

$$E[X] = \frac◆LB◆a + b◆RB◆◆LB◆2◆RB◆, \quad \mathrm◆LB◆Var◆RB◆(X) = \frac◆LB◆(b-a)^2◆RB◆◆LB◆12◆RB◆$$

$$M_X(t) = \begin◆LB◆cases◆RB◆ \frac◆LB◆e^◆LB◆tb◆RB◆ - e^◆LB◆ta◆RB◆◆RB◆◆LB◆t(b - a)◆RB◆ & \mathrm◆LB◆if◆RB◆  t \neq 0 \\ 1 & \mathrm◆LB◆if◆RB◆  t = 0 \end◆LB◆cases◆RB◆$$

**Gamma.** $X \sim \mathrm◆LB◆Gamma◆RB◆(\alpha, \lambda)$ (shape $\alpha \gt 0$Rate $\lambda \gt 0$):

$$f(x) = \frac◆LB◆\lambda^\alpha◆RB◆◆LB◆\Gamma(\alpha)◆RB◆ x^◆LB◆\alpha - 1◆RB◆ e^◆LB◆-\lambda x◆RB◆, \quad x \gt 0$$

Where $\Gamma(\alpha) = \int_0^\infty t^◆LB◆\alpha - 1◆RB◆ e^◆LB◆-t◆RB◆\, dt$ is the Gamma function.

$$E[X] = \frac◆LB◆\alpha◆RB◆◆LB◆\lambda◆RB◆, \quad \mathrm◆LB◆Var◆RB◆(X) = \frac◆LB◆\alpha◆RB◆◆LB◆\lambda^2◆RB◆$$

$$M_X(t) = \left(\frac◆LB◆\lambda◆RB◆◆LB◆\lambda - t◆RB◆\right)^\alpha, \quad t \lt \lambda$$

_Remark._ Special cases: $\mathrm◆LB◆Gamma◆RB◆(1, \lambda) = \mathrm◆LB◆Exp◆RB◆(\lambda)$;
$\mathrm◆LB◆Gamma◆RB◆(n/2, 1/2) = \chi^2_n$.

**Theorem 3.0 (Sum of Independent Gammas).** If $X \sim \mathrm◆LB◆Gamma◆RB◆(\alpha_1, \lambda)$ and
$Y \sim \mathrm◆LB◆Gamma◆RB◆(\alpha_2, \lambda)$ are independent, then
$X + Y \sim \mathrm◆LB◆Gamma◆RB◆(\alpha_1 + \alpha_2, \lambda)$.

_Proof._
$M_◆LB◆X+Y◆RB◆(t) = M_X(t)\, M_Y(t) = \left(\frac◆LB◆\lambda◆RB◆◆LB◆\lambda - t◆RB◆\right)^◆LB◆\alpha_1◆RB◆ \left(\frac◆LB◆\lambda◆RB◆◆LB◆\lambda - t◆RB◆\right)^◆LB◆\alpha_2◆RB◆
= \left(\frac◆LB◆\lambda◆RB◆◆LB◆\lambda - t◆RB◆\right)^◆LB◆\alpha_1 + \alpha_2◆RB◆$,
which is the MGF of $\mathrm◆LB◆Gamma◆RB◆(\alpha_1 + \alpha_2, \lambda)$. $\blacksquare$

**Chi-Squared.** $X \sim \chi^2_k$ (chi-squared with $k$ degrees of freedom):

$$f(x) = \frac◆LB◆1◆RB◆◆LB◆2^◆LB◆k/2◆RB◆\, \Gamma(k/2)◆RB◆\, x^◆LB◆k/2 - 1◆RB◆\, e^◆LB◆-x/2◆RB◆, \quad x \gt 0$$

$$E[X] = k, \quad \mathrm◆LB◆Var◆RB◆(X) = 2k$$

$$M_X(t) = (1 - 2t)^◆LB◆-k/2◆RB◆, \quad t \lt 1/2$$

_Remark._ If $Z_1, \ldots, Z_k \sim N(0, 1)$ are independent, then
$\sum_◆LB◆i=1◆RB◆^k Z_i^2 \sim \chi^2_k$.

**Beta.** $X \sim \mathrm◆LB◆Beta◆RB◆(\alpha, \beta)$:

$$f(x) = \frac◆LB◆x^◆LB◆\alpha - 1◆RB◆(1 - x)^◆LB◆\beta - 1◆RB◆◆RB◆◆LB◆B(\alpha, \beta)◆RB◆, \quad 0 \lt x \lt 1$$

Where $B(\alpha, \beta) = \frac◆LB◆\Gamma(\alpha)\Gamma(\beta)◆RB◆◆LB◆\Gamma(\alpha + \beta)◆RB◆$ is the Beta
function.

$$E[X] = \frac◆LB◆\alpha◆RB◆◆LB◆\alpha + \beta◆RB◆, \quad \mathrm◆LB◆Var◆RB◆(X) = \frac◆LB◆\alpha \beta◆RB◆◆LB◆(\alpha + \beta)^2(\alpha + \beta + 1)◆RB◆$$

**Exponential.** $X \sim \mathrm◆LB◆Exp◆RB◆(\lambda)$:

$$f(x) = \begin◆LB◆cases◆RB◆ \lambda e^◆LB◆-\lambda x◆RB◆ & \mathrm◆LB◆if◆RB◆  x \geq 0 \\ 0 & \mathrm◆LB◆if◆RB◆  x \lt 0 \end◆LB◆cases◆RB◆$$

$$E[X] = \frac◆LB◆1◆RB◆◆LB◆\lambda◆RB◆, \quad \mathrm◆LB◆Var◆RB◆(X) = \frac◆LB◆1◆RB◆◆LB◆\lambda^2◆RB◆$$

**Theorem 3.1 (Memoryless Property).** If $X \sim \mathrm◆LB◆Exp◆RB◆(\lambda)$Then for all $s, t \gt 0$:

$$P(X \gt s + t \mid X \gt s) = P(X \gt t)$$

_Proof._

$$P(X \gt s + t \mid X \gt s) = \frac◆LB◆P(X \gt s + t)◆RB◆◆LB◆P(X \gt s)◆RB◆ = \frac◆LB◆e^◆LB◆-\lambda(s+t)◆RB◆◆RB◆◆LB◆e^◆LB◆-\lambda s◆RB◆◆RB◆ = e^◆LB◆-\lambda t◆RB◆ = P(X \gt t)$$

$\blacksquare$

_Remark._ The exponential distribution is the **only** continuous distribution with the memoryless
property. This Makes it the natural model for waiting times between Poisson events.

**Normal (Gaussian).** $X \sim N(\mu, \sigma^2)$:

$$f(x) = \frac◆LB◆1◆RB◆◆LB◆\sigma\sqrt◆LB◆2\pi◆RB◆◆RB◆ \exp\left(-\frac◆LB◆(x - \mu)^2◆RB◆◆LB◆2\sigma^2◆RB◆\right), \quad x \in \mathbb◆LB◆R◆RB◆$$

$$E[X] = \mu, \quad \mathrm◆LB◆Var◆RB◆(X) = \sigma^2$$

The **standard normal** $Z \sim N(0,1)$ has CDF denoted $\Phi(z)$. For any
$X \sim N(\mu, \sigma^2)$:

$$Z = \frac◆LB◆X - \mu◆RB◆◆LB◆\sigma◆RB◆ \sim N(0, 1)$$

$$M_X(t) = \exp\left(\mu t + \frac◆LB◆\sigma^2 t^2◆RB◆◆LB◆2◆RB◆\right)$$

_Verification that $f$ integrates to 1._ Consider $I = \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ e^◆LB◆-x^2/2◆RB◆\, dx$.
Then $I^2 = \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆\int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ e^◆LB◆-(x^2+y^2)/2◆RB◆\, dx\, dy$. Switching to
polar coordinates: $x = r\cos\theta$$y = r\sin\theta$$dx\, dy = r\, dr\, d\theta$.

$$I^2 = \int_0^◆LB◆2\pi◆RB◆\int_0^◆LB◆\infty◆RB◆ e^◆LB◆-r^2/2◆RB◆\, r\, dr\, d\theta = 2\pi \int_0^◆LB◆\infty◆RB◆ e^◆LB◆-r^2/2◆RB◆\, r\, dr = 2\pi \left[-e^◆LB◆-r^2/2◆RB◆\right]_0^◆LB◆\infty◆RB◆ = 2\pi$$

So $I = \sqrt◆LB◆2\pi◆RB◆$Confirming that
$\frac◆LB◆1◆RB◆◆LB◆\sqrt◆LB◆2\pi◆RB◆◆RB◆\int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ e^◆LB◆-x^2/2◆RB◆\, dx = 1$. For the general
$N(\mu, \sigma^2)$The substitution $z = (x - \mu)/\sigma$ reduces to the standard case.

_Verification that $E[X] = \mu$ for $Z \sim N(0, 1)$:_ The integrand $z \cdot \phi(z)$ is an odd
function of $z$So $\int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ z\, \phi(z)\, dz = 0$. For $X = \mu + \sigma Z$:
$E[X] = \mu + \sigma \cdot 0 = \mu$.

_Verification that $\mathrm◆LB◆Var◆RB◆(Z) = 1$:_ Integration by parts with $u = z$, $dv = z\, \phi(z)\, dz$:

$$E[Z^2] = \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ z^2 \phi(z)\, dz = \left[-z\phi(z)\right]_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ + \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ \phi(z)\, dz = 0 + 1 = 1$$

_Proof of MGF for $Z \sim N(0, 1)$:_

$$M_Z(t) = \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ e^◆LB◆tz◆RB◆ \frac◆LB◆1◆RB◆◆LB◆\sqrt◆LB◆2\pi◆RB◆◆RB◆ e^◆LB◆-z^2/2◆RB◆\, dz = \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ \frac◆LB◆1◆RB◆◆LB◆\sqrt◆LB◆2\pi◆RB◆◆RB◆ e^◆LB◆-(z^2 - 2tz)/2◆RB◆\, dz$$

Completing the square: $z^2 - 2tz = (z - t)^2 - t^2$So:

$$M_Z(t) = e^◆LB◆t^2/2◆RB◆ \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ \frac◆LB◆1◆RB◆◆LB◆\sqrt◆LB◆2\pi◆RB◆◆RB◆ e^◆LB◆-(z-t)^2/2◆RB◆\, dz = e^◆LB◆t^2/2◆RB◆ \cdot 1 = e^◆LB◆t^2/2◆RB◆$$

Since the integrand is the PDF of $N(t, 1)$ evaluated over all $\mathbb◆LB◆R◆RB◆$. $\blacksquare$

For $X = \mu + \sigma Z$:
$M_X(t) = E[e^◆LB◆t(\mu + \sigma Z)◆RB◆] = e^◆LB◆\mu t◆RB◆ E[e^◆LB◆(\sigma t)Z◆RB◆] = e^◆LB◆\mu t◆RB◆ e^◆LB◆\sigma^2 t^2/2◆RB◆$.

**Theorem 3.2.** The sum of independent normal random variables is normal: If
$X_i \sim N(\mu_i, \sigma_i^2)$ are independent, then
$\sum X_i \sim N(\sum \mu_i, \sum \sigma_i^2)$.

_Proof (using MGFs)._
$M_◆LB◆\sum X_i◆RB◆(t) = \prod_i M_◆LB◆X_i◆RB◆(t) = \prod_i \exp(\mu_i t + \sigma_i^2 t^2/2)
= \exp\left((\sum \mu_i)t + \frac◆LB◆(\sum \sigma_i^2) t^2◆RB◆◆LB◆2◆RB◆\right)$,
which is the MGF of $N(\sum \mu_i, \sum \sigma_i^2)$. By the uniqueness theorem for MGFs, the result
follows. $\blacksquare$

### 3.3 Relationships Between Distributions

**Theorem 3.3 (Poisson Limit Theorem).** If $X_n \sim \mathrm◆LB◆Bin◆RB◆(n, p_n)$ where $np_n \to \lambda$
as $n \to \infty$ Then $X_n \xrightarrow◆LB◆d◆RB◆ \mathrm◆LB◆Poisson◆RB◆(\lambda)$.

_Proof._ Let $\lambda_n = np_n$ so that $p_n = \lambda_n / n$. Then:

$$P(X_n = k) = \binom◆LB◆n◆RB◆◆LB◆k◆RB◆ p_n^k (1 - p_n)^◆LB◆n-k◆RB◆ = \frac◆LB◆n(n-1)\cdots(n-k+1)◆RB◆◆LB◆k!◆RB◆ \left(\frac◆LB◆\lambda_n◆RB◆◆LB◆n◆RB◆\right)^k \left(1 - \frac◆LB◆\lambda_n◆RB◆◆LB◆n◆RB◆\right)^◆LB◆n-k◆RB◆$$

$$= \frac◆LB◆\lambda_n^k◆RB◆◆LB◆k!◆RB◆ \cdot \frac◆LB◆n◆RB◆◆LB◆n◆RB◆ \cdot \frac◆LB◆n-1◆RB◆◆LB◆n◆RB◆ \cdots \frac◆LB◆n-k+1◆RB◆◆LB◆n◆RB◆ \cdot \left(1 - \frac◆LB◆\lambda_n◆RB◆◆LB◆n◆RB◆\right)^n \cdot \left(1 - \frac◆LB◆\lambda_n◆RB◆◆LB◆n◆RB◆\right)^◆LB◆-k◆RB◆$$

As $n \to \infty$: $\lambda_n \to \lambda$; $\frac◆LB◆n-j◆RB◆◆LB◆n◆RB◆ \to 1$ for each fixed $j$;
$\left(1 - \frac◆LB◆\lambda_n◆RB◆◆LB◆n◆RB◆\right)^n \to e^◆LB◆-\lambda◆RB◆$; and
$\left(1 - \frac◆LB◆\lambda_n◆RB◆◆LB◆n◆RB◆\right)^◆LB◆-k◆RB◆ \to 1$.

Therefore: $P(X_n = k) \to \frac◆LB◆\lambda^k e^◆LB◆-\lambda◆RB◆◆RB◆◆LB◆k!◆RB◆ = P(\mathrm◆LB◆Poisson◆RB◆(\lambda) = k)$.
$\blacksquare$

_Intuition._ The Poisson distribution approximates the binomial when $n$ is large, $p$ is small, and
$np$ is moderate.

**Theorem 3.4 (Normal approximation to the Binomial).** If $X \sim \mathrm◆LB◆Bin◆RB◆(n, p)$ with $n$
large, then Approximately $X \approx N(np, np(1-p))$. More precisely, using a continuity correction:

$$P(a \leq X \leq b) \approx \Phi\left(\frac◆LB◆b + 0.5 - np◆RB◆◆LB◆\sqrt◆LB◆np(1-p)◆RB◆◆RB◆\right) - \Phi\left(\frac◆LB◆a - 0.5 - np◆RB◆◆LB◆\sqrt◆LB◆np(1-p)◆RB◆◆RB◆\right)$$

_Intuition._ By the CLT, the sum of $n$ i.i.d. $\mathrm◆LB◆Bernoulli◆RB◆(p)$ variables (each with mean $p$
and variance $p(1-p)$) is approximately normal.

## 4. Expectation, Variance, and Moment Generating Functions

### 4.1 Expectation

**Definition.** The **expected value** of $X$ is

$$E[X] = \begin◆LB◆cases◆RB◆ \sum_x x\, f_X(x) & \mathrm◆LB◆(discrete)◆RB◆ \\ \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ x\, f_X(x)\, dx & \mathrm◆LB◆(continuous)◆RB◆ \end◆LB◆cases◆RB◆$$

**Proposition 4.1 (LOTUS — Law of the Unconscious Statistician).** For any function $g$:

$$E[g(X)] = \begin◆LB◆cases◆RB◆ \sum_x g(x)\, f_X(x) & \mathrm◆LB◆(discrete)◆RB◆ \\ \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ g(x)\, f_X(x)\, dx & \mathrm◆LB◆(continuous)◆RB◆ \end◆LB◆cases◆RB◆$$

_Intuition._ We do not need to find the distribution of $Y = g(X)$ to compute $E[Y]$; we integrate
with respect to the Distribution of $X$ directly.

**Theorem 4.1 (Linearity of Expectation).**

1. $E[aX + b] = aE[X] + b$.
2. $E[X + Y] = E[X] + E[Y]$ for any random variables $X, Y$ (no independence required).

_Proof._ We prove (2) for the continuous case; the discrete case is analogous.

$$E[X + Y] = \iint_◆LB◆\mathbb◆LB◆R◆RB◆^2◆RB◆ (x + y)\, f_◆LB◆X,Y◆RB◆(x,y)\, dx\, dy$$

$$= \iint_◆LB◆\mathbb◆LB◆R◆RB◆^2◆RB◆ x\, f_◆LB◆X,Y◆RB◆(x,y)\, dx\, dy + \iint_◆LB◆\mathbb◆LB◆R◆RB◆^2◆RB◆ y\, f_◆LB◆X,Y◆RB◆(x,y)\, dx\, dy$$

$$= \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ x \left(\int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ f_◆LB◆X,Y◆RB◆(x,y)\, dy\right) dx + \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ y \left(\int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ f_◆LB◆X,Y◆RB◆(x,y)\, dx\right) dy$$

$$= \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ x\, f_X(x)\, dx + \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ y\, f_Y(y)\, dy = E[X] + E[Y] \quad \blacksquare$$

3. If $X$ and $Y$ are independent, $E[XY] = E[X]E[Y]$.

_Proof._
$E[XY] = \iint xy\, f_X(x)f_Y(y)\, dx\, dy = \left(\int x f_X(x)\, dx\right)\left(\int y f_Y(y)\, dy\right)
= E[X]E[Y]$.
$\blacksquare$

### 4.2 Variance

$$\mathrm◆LB◆Var◆RB◆(X) = E[(X - E[X])^2] = E[X^2] - (E[X])^2$$

**Theorem 4.2.**

1. $\mathrm◆LB◆Var◆RB◆(aX + b) = a^2 \mathrm◆LB◆Var◆RB◆(X)$.
2. If $X, Y$ are independent: $\mathrm◆LB◆Var◆RB◆(X + Y) = \mathrm◆LB◆Var◆RB◆(X) + \mathrm◆LB◆Var◆RB◆(Y)$.

_Proof._ (1)
$\mathrm◆LB◆Var◆RB◆(aX + b) = E[(aX + b - aE[X] - b)^2] = E[a^2(X - E[X])^2] = a^2 \mathrm◆LB◆Var◆RB◆(X)$.

(2) $\mathrm◆LB◆Var◆RB◆(X + Y) = E[(X + Y)^2] - (E[X] + E[Y])^2$
$= E[X^2] + 2E[XY] + E[Y^2] - E[X]^2 - 2E[X]E[Y] - E[Y]^2$
$= (E[X^2] - E[X]^2) + (E[Y^2] - E[Y]^2) + 2(E[XY] - E[X]E[Y])$
$= \mathrm◆LB◆Var◆RB◆(X) + \mathrm◆LB◆Var◆RB◆(Y) + 2\,\mathrm◆LB◆Cov◆RB◆(X,Y)$.

If $X, Y$ are independent, $\mathrm◆LB◆Cov◆RB◆(X,Y) = 0$. $\blacksquare$

### 4.3 Moment Generating Functions

The **moment generating function** (MGF) of $X$ is

$$M_X(t) = E[e^◆LB◆tX◆RB◆]$$

(provided the expectation exists in a neighbourhood of $t = 0$).

**Theorem 4.3.** If $M_X(t)$ exists in a neighbourhood of $0$Then $E[X^n] = M_X^◆LB◆(n)◆RB◆(0)$.

_Proof._ $M_X(t) = E[e^◆LB◆tX◆RB◆] = \sum_◆LB◆n=0◆RB◆^◆LB◆\infty◆RB◆ \frac◆LB◆t^n◆RB◆◆LB◆n!◆RB◆ E[X^n]$ (by expanding the Taylor
series and exchanging Summation and expectation, justified by dominated convergence). The
coefficient of $t^n/n!$ is $E[X^n]$So $E[X^n] = M_X^◆LB◆(n)◆RB◆(0)$. $\blacksquare$

**Theorem 4.4 (Uniqueness).** If $M_X(t) = M_Y(t)$ for all $t$ in a neighbourhood of $0$Then $X$ And
$Y$ have the same distribution.

**Theorem 4.5.** If $X$ and $Y$ are independent, $M_◆LB◆X+Y◆RB◆(t) = M_X(t) M_Y(t)$.

_Proof._ $M_◆LB◆X+Y◆RB◆(t) = E[e^◆LB◆t(X+Y)◆RB◆] = E[e^◆LB◆tX◆RB◆ e^◆LB◆tY◆RB◆] = E[e^◆LB◆tX◆RB◆]\, E[e^◆LB◆tY◆RB◆] = M_X(t)\, M_Y(t)$
Where the third equality uses independence. $\blacksquare$

### 4.4 Important Inequalities

**Theorem 4.6a (Markov's Inequality).** If $X \geq 0$ and $a \gt 0$:

$$P(X \geq a) \leq \frac◆LB◆E[X]◆RB◆◆LB◆a◆RB◆$$

_Proof._
$E[X] = \int_0^\infty x\, dF(x) \geq \int_a^\infty x\, dF(x) \geq a \int_a^\infty dF(x) = a\, P(X \geq a)$.
$\blacksquare$

**Theorem 4.6b (Chebyshev's Inequality).** For any random variable $X$ with finite mean $\mu$ and
variance $\sigma^2$ And any $k \gt 0$:

$$P(|X - \mu| \geq k) \leq \frac◆LB◆\sigma^2◆RB◆◆LB◆k^2◆RB◆$$

_Proof._ Apply Markov's inequality to $(X - \mu)^2$ with $a = k^2$:
$P(|X - \mu| \geq k) = P((X - \mu)^2 \geq k^2) \leq \frac◆LB◆E[(X-\mu)^2]◆RB◆◆LB◆k^2◆RB◆ = \frac◆LB◆\sigma^2◆RB◆◆LB◆k^2◆RB◆$.
$\blacksquare$

**Theorem 4.6c (Jensen's Inequality).** If $\varphi$ is convex, then
$E[\varphi(X)] \geq \varphi(E[X])$. If $\varphi$ is concave, the inequality reverses.

_Proof (sketch)._ For a convex function $\varphi$The tangent line at any point lies below the graph:
$\varphi(x) \geq \varphi(\mu) + \varphi'(\mu)(x - \mu)$ where $\mu = E[X]$. Taking expectations of
both sides: $E[\varphi(X)] \geq \varphi(\mu) + \varphi'(\mu) \cdot 0 = \varphi(E[X])$.
$\blacksquare$

_Remark._ Important applications: $E[X^2] \geq (E[X])^2$ (variance is non-negative, since $x^2$ is
convex); $E[\log X] \leq \log E[X]$ (logarithm is concave — this is used in proving the information
inequality).

### 4.5 Cauchy-Schwarz Inequality for Random Variables

**Theorem 4.6 (Cauchy-Schwarz).** For any random variables $X, Y$ with finite second moments:

$$(E[XY])^2 \leq E[X^2]\, E[Y^2]$$

_Proof._ For any real $t$, $E[(X + tY)^2] = E[X^2] + 2t\, E[XY] + t^2\, E[Y^2] \geq 0$. This is a
quadratic in $t$ That is always non-negative, so its discriminant must be non-positive:

$$(2E[XY])^2 - 4\, E[Y^2]\, E[X^2] \leq 0 \implies (E[XY])^2 \leq E[X^2]\, E[Y^2]$$

$\blacksquare$

**Corollary 4.1.** $|\rho_◆LB◆X,Y◆RB◆| \leq 1$.

_Proof._ Apply Cauchy-Schwarz to $X - E[X]$ and $Y - E[Y]$:

$$\mathrm◆LB◆Cov◆RB◆(X,Y)^2 \leq \mathrm◆LB◆Var◆RB◆(X)\, \mathrm◆LB◆Var◆RB◆(Y)$$

So $|\rho_◆LB◆X,Y◆RB◆| = \frac◆LB◆|\mathrm◆LB◆Cov◆RB◆(X,Y)|◆RB◆◆LB◆\sqrt◆LB◆\mathrm◆LB◆Var◆RB◆(X)\,\mathrm◆LB◆Var◆RB◆(Y)◆RB◆◆RB◆ \leq 1$.
$\blacksquare$

**Theorem 4.7 (Conditional Variance Formula).**

$$\mathrm◆LB◆Var◆RB◆(Y) = E[\mathrm◆LB◆Var◆RB◆(Y \mid X)] + \mathrm◆LB◆Var◆RB◆(E[Y \mid X])$$

_Proof._
$E[\mathrm◆LB◆Var◆RB◆(Y \mid X)] = E[E[Y^2 \mid X]] - E[(E[Y \mid X])^2] = E[Y^2] - E[(E[Y \mid X])^2]$.
Also
$\mathrm◆LB◆Var◆RB◆(E[Y \mid X]) = E[(E[Y \mid X])^2] - (E[E[Y \mid X]])^2 = E[(E[Y \mid X])^2] - (E[Y])^2$.
Adding: $E[Y^2] - (E[Y])^2 = \mathrm◆LB◆Var◆RB◆(Y)$. $\blacksquare$

_Intuition._ Total variance equals the average "within-group" variance plus the "between-group"
variance. This Decomposition is the foundation of ANOVA (Analysis of Variance).

## 5. Joint Distributions

### 5.1 Joint PDF and CDF

For two random variables $X$ and $Y$The **joint CDF** is $F_◆LB◆X,Y◆RB◆(x,y) = P(X \leq x, Y \leq y)$.

The **joint PDF** (for continuous) satisfies $P((X,Y) \in A) = \iint_A f_◆LB◆X,Y◆RB◆(x,y)\, dx\, dy$.

### 5.2 Marginal Distributions

The **marginal PDF** of $X$ is obtained by integrating out $Y$:

$$f_X(x) = \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ f_◆LB◆X,Y◆RB◆(x,y)\, dy$$

Similarly, $f_Y(y) = \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ f_◆LB◆X,Y◆RB◆(x,y)\, dx$.

### 5.3 Conditional Distributions

**Definition.** The **conditional PDF** of $X$ given $Y = y$ (when $f_Y(y) \gt 0$) is

$$f_◆LB◆X \mid Y◆RB◆(x \mid y) = \frac◆LB◆f_◆LB◆X,Y◆RB◆(x,y)◆RB◆◆LB◆f_Y(y)◆RB◆$$

**Definition.** The **conditional expectation** is

$$E[X \mid Y = y] = \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ x\, f_◆LB◆X \mid Y◆RB◆(x \mid y)\, dx$$

**Theorem 5.0 (Law of Iterated Expectations / Tower Property).**

$$E[X] = E[E[X \mid Y]]$$

_Proof._ For the continuous case:

$$E[E[X \mid Y]] = \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ E[X \mid Y = y]\, f_Y(y)\, dy = \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ \left(\int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ x\, f_◆LB◆X \mid Y◆RB◆(x \mid y)\, dx\right) f_Y(y)\, dy$$

$$= \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ x\, \frac◆LB◆f_◆LB◆X,Y◆RB◆(x,y)◆RB◆◆LB◆f_Y(y)◆RB◆\, f_Y(y)\, dx\, dy = \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ \int_◆LB◆-\infty◆RB◆^◆LB◆\infty◆RB◆ x\, f_◆LB◆X,Y◆RB◆(x,y)\, dx\, dy = E[X]$$

$\blacksquare$

### 5.4 Independence

$X$ and $Y$ are **independent** if and only if

$$f_◆LB◆X,Y◆RB◆(x,y) = f_X(x) f_Y(y) \quad \mathrm◆LB◆for◆RB◆ all  x, y$$

**Theorem 5.1.** If $X$ and $Y$ are independent, then $E[XY] = E[X]E[Y]$ and
$\mathrm◆LB◆Var◆RB◆(X + Y) = \mathrm◆LB◆Var◆RB◆(X) + \mathrm◆LB◆Var◆RB◆(Y)$.

**Proposition 5.1 (Independence Criteria).** The following are equivalent for continuous $X, Y$:

1. $X$ and $Y$ are independent.
2. $f_◆LB◆X,Y◆RB◆(x,y) = f_X(x) f_Y(y)$ for all $x, y$.
3. $f_◆LB◆X \mid Y◆RB◆(x \mid y) = f_X(x)$ for all $x, y$ with $f_Y(y) \gt 0$.
4. $F_◆LB◆X,Y◆RB◆(x,y) = F_X(x)\, F_Y(y)$ for all $x, y$.

### 5.5 Covariance and Correlation

$$\mathrm◆LB◆Cov◆RB◆(X, Y) = E[(X - E[X])(Y - E[Y])] = E[XY] - E[X]E[Y]$$

The **correlation coefficient** is

$$\rho_◆LB◆X,Y◆RB◆ = \frac◆LB◆\mathrm◆LB◆Cov◆RB◆(X,Y)◆RB◆◆LB◆\sqrt◆LB◆\mathrm◆LB◆Var◆RB◆(X)\mathrm◆LB◆Var◆RB◆(Y)◆RB◆◆RB◆$$

**Properties:**

- $\mathrm◆LB◆Cov◆RB◆(aX + b, cY + d) = ac\,\mathrm◆LB◆Cov◆RB◆(X, Y)$.
- $\mathrm◆LB◆Cov◆RB◆(X, X) = \mathrm◆LB◆Var◆RB◆(X)$.
- $\mathrm◆LB◆Cov◆RB◆(X + Z, Y) = \mathrm◆LB◆Cov◆RB◆(X, Y) + \mathrm◆LB◆Cov◆RB◆(Z, Y)$ (bilinearity).
- $-1 \leq \rho_◆LB◆X,Y◆RB◆ \leq 1$.
- $\rho = \pm 1$ if and only if $X$ and $Y$ are linearly related.
- $\rho = 0$ does not imply independence (only uncorrelatedness).

**Proposition 5.2 (Variance of a Sum).** For any random variables $X_1, \ldots, X_n$:

$$\mathrm◆LB◆Var◆RB◆\left(\sum_◆LB◆i=1◆RB◆^n X_i\right) = \sum_◆LB◆i=1◆RB◆^n \mathrm◆LB◆Var◆RB◆(X_i) + 2\sum_◆LB◆1 \leq i \lt j \leq n◆RB◆ \mathrm◆LB◆Cov◆RB◆(X_i, X_j)$$

_Proof._ Expand $\mathrm◆LB◆Var◆RB◆(\sum X_i) = E[(\sum X_i)^2] - (E[\sum X_i])^2$ and collect terms.
$\blacksquare$

_Remark._ If the $X_i$ are pairwise uncorrelated (which includes independence as a special case),
the covariance Terms vanish and the variance of the sum equals the sum of the variances.

### 5.6 Transformation of Random Variables (Jacobian Method)

**Theorem 5.2.** Let $(X, Y)$ have joint PDF $f_◆LB◆X,Y◆RB◆(x,y)$ and let $(U, V) = g(X, Y)$ where $g$ is
a bijection From $\mathbb◆LB◆R◆RB◆^2$ to $\mathbb◆LB◆R◆RB◆^2$ with inverse $g^◆LB◆-1◆RB◆$:

$$u = u(x, y), \quad v = v(x, y)$$

Then the joint PDF of $(U, V)$ is

$$f_◆LB◆U,V◆RB◆(u,v) = f_◆LB◆X,Y◆RB◆(x(u,v), y(u,v)) \cdot |J|$$

Where the **Jacobian determinant** is

$$J = \det \begin◆LB◆pmatrix◆RB◆ \frac◆LB◆\partial x◆RB◆◆LB◆\partial u◆RB◆ & \frac◆LB◆\partial x◆RB◆◆LB◆\partial v◆RB◆ \\ \frac◆LB◆\partial y◆RB◆◆LB◆\partial u◆RB◆ & \frac◆LB◆\partial y◆RB◆◆LB◆\partial v◆RB◆ \end◆LB◆pmatrix◆RB◆$$

**Problem 5.1.** Let $X, Y$ be independent with $X, Y \sim \mathrm◆LB◆Exp◆RB◆(\lambda)$. Find the joint
distribution of $U = X + Y$ and $V = X / (X + Y)$.

<details>
<summary>Solution</summary>

The inverse transformation is $X = UV$, $Y = U(1 - V)$ for $u \gt 0, 0 \lt v \lt 1$.

The Jacobian:

$$J = \det \begin◆LB◆pmatrix◆RB◆ v & u \\ 1 - v & -u \end◆LB◆pmatrix◆RB◆ = -uv - u(1 - v) = -u$$

So $|J| = u$.

The joint PDF of $(X, Y)$ is $f_◆LB◆X,Y◆RB◆(x,y) = \lambda^2 e^◆LB◆-\lambda(x+y)◆RB◆$ for $x, y \gt 0$.

$$f_◆LB◆U,V◆RB◆(u,v) = \lambda^2 e^◆LB◆-\lambda(uv + u(1-v))◆RB◆ \cdot u = \lambda^2 u\, e^◆LB◆-\lambda u◆RB◆, \quad u \gt 0, 0 \lt v \lt 1$$

This factors as $f_U(u) \cdot f_V(v)$ where $f_U(u) = \lambda^2 u\, e^◆LB◆-\lambda u◆RB◆$
(Gamma$(2, \lambda)$) And $f_V(v) = 1$ on $(0, 1)$ (Uniform$(0, 1)$). Hence $U$ and $V$ are
independent. $\blacksquare$

</details>

### 5.7 Bivariate Normal Distribution

**Definition.** $(X, Y)$ has a **bivariate normal distribution** with parameters
$\mu_X, \mu_Y, \sigma_X^2, \sigma_Y^2, \rho$ If the joint PDF is

$$f(x,y) = \frac◆LB◆1◆RB◆◆LB◆2\pi \sigma_X \sigma_Y \sqrt◆LB◆1 - \rho^2◆RB◆◆RB◆ \exp\left(-\frac◆LB◆1◆RB◆◆LB◆2(1-\rho^2)◆RB◆\left[\frac◆LB◆(x - \mu_X)^2◆RB◆◆LB◆\sigma_X^2◆RB◆ - \frac◆LB◆2\rho(x - \mu_X)(y - \mu_Y)◆RB◆◆LB◆\sigma_X \sigma_Y◆RB◆ + \frac◆LB◆(y - \mu_Y)^2◆RB◆◆LB◆\sigma_Y^2◆RB◆\right]\right)$$

**Key Properties:**

1. Both marginals are normal: $X \sim N(\mu_X, \sigma_X^2)$ and $Y \sim N(\mu_Y, \sigma_Y^2)$.
2. $\rho = \mathrm◆LB◆Corr◆RB◆(X, Y)$.
3. $X$ and $Y$ are independent if and only if $\rho = 0$.
4. Every linear combination $aX + bY$ is normally distributed.
5. The conditional distribution $Y \mid X = x$ is normal with
   $E[Y \mid X = x] = \mu_Y + \rho \frac◆LB◆\sigma_Y◆RB◆◆LB◆\sigma_X◆RB◆(x - \mu_X)$ and
   $\mathrm◆LB◆Var◆RB◆(Y \mid X = x) = \sigma_Y^2(1 - \rho^2)$.
6. The joint MGF is

$$M_◆LB◆X,Y◆RB◆(t_1, t_2) = \exp\left(\mu_X t_1 + \mu_Y t_2 + \frac◆LB◆1◆RB◆◆LB◆2◆RB◆(\sigma_X^2 t_1^2 + 2\rho\sigma_X\sigma_Y t_1 t_2 + \sigma_Y^2 t_2^2)\right)$$

**Problem 5.4.** Let $(X, Y)$ be bivariate normal with

## 6. Limit Theorems

### 6.1 Convergence in Probability and Distribution

**Definition.** $X_n \xrightarrow◆LB◆p◆RB◆ X$ (convergence in probability) if for every
$\varepsilon \gt 0$:

$$\lim_◆LB◆n \to \infty◆RB◆ P(|X_n - X| \gt \varepsilon) = 0$$

**Definition.** $X_n \xrightarrow◆LB◆d◆RB◆ X$ (convergence in distribution) if
$\lim_◆LB◆n \to \infty◆RB◆ F_◆LB◆X_n◆RB◆(x) = F_X(x)$ at all Continuity points of $F_X$.

_Remark._ Convergence in probability implies convergence in distribution. The converse does not hold
, But does hold when the limit is a constant.

**Proposition 6.1.** If $X_n \xrightarrow◆LB◆p◆RB◆ c$ (a constant), then $X_n \xrightarrow◆LB◆d◆RB◆ c$.

_Proof._ If $F_c(x)$ is the CDF of the constant $c$Then $F_c(x) = 0$ for $x \lt c$ and $F_c(x) = 1$
for $x \gt c$. For $x \gt c$:
$F_◆LB◆X_n◆RB◆(x) = P(X_n \leq x) = 1 - P(X_n \gt x) \to 1 - 0 = 1 = F_c(x)$. For $x \lt c$:
$F_◆LB◆X_n◆RB◆(x) = P(X_n \leq x) \leq P(|X_n - c| \gt c - x) \to 0 = F_c(x)$. Since $F_c$ is continuous
at all $x \neq c$Convergence holds. $\blacksquare$

**Definition.** $X_n \xrightarrow◆LB◆a.s.◆RB◆ X$ (almost sure convergence) if
$P(\lim_◆LB◆n \to \infty◆RB◆ X_n = X) = 1$.

_Remark._ The hierarchy of convergence is: almost sure $\implies$ in probability $\implies$ in
distribution. None of the reverse implications hold .

### 6.2 Law of Large Numbers

**Theorem 6.1 (Weak Law of Large Numbers).** Let $X_1, X_2, \ldots$ be i.i.d. With $E[X_i] = \mu$
and $\mathrm◆LB◆Var◆RB◆(X_i) = \sigma^2 \lt \infty$. Then for every $\varepsilon \gt 0$:

$$\lim_◆LB◆n \to \infty◆RB◆ P\left(\left|\frac◆LB◆1◆RB◆◆LB◆n◆RB◆\sum_◆LB◆i=1◆RB◆^n X_i - \mu\right| \gt \varepsilon\right) = 0$$

_Proof._ Let $\bar◆LB◆X◆RB◆_n = \frac◆LB◆1◆RB◆◆LB◆n◆RB◆\sum_◆LB◆i=1◆RB◆^n X_i$. Then $E[\bar◆LB◆X◆RB◆_n] = \mu$ and
$\mathrm◆LB◆Var◆RB◆(\bar◆LB◆X◆RB◆_n) = \sigma^2/n$. By Chebyshev's inequality:

$$P(|\bar◆LB◆X◆RB◆_n - \mu| \gt \varepsilon) \leq \frac◆LB◆\mathrm◆LB◆Var◆RB◆(\bar◆LB◆X◆RB◆_n)◆RB◆◆LB◆\varepsilon^2◆RB◆ = \frac◆LB◆\sigma^2◆RB◆◆LB◆n\varepsilon^2◆RB◆ \to 0$$

$\blacksquare$

**Theorem 6.2 (Strong Law of Large Numbers).** Under the same hypotheses:

$$P\left(\lim_◆LB◆n \to \infty◆RB◆ \frac◆LB◆1◆RB◆◆LB◆n◆RB◆\sum_◆LB◆i=1◆RB◆^n X_i = \mu\right) = 1$$

### 6.3 Central Limit Theorem

**Theorem 6.3 (Central Limit Theorem).** Let $X_1, X_2, \ldots$ be i.i.d. With $E[X_i] = \mu$ and
$\mathrm◆LB◆Var◆RB◆(X_i) = \sigma^2 \gt 0$. Then

$$\frac◆LB◆\bar◆LB◆X◆RB◆_n - \mu◆RB◆◆LB◆\sigma / \sqrt◆LB◆n◆RB◆◆RB◆ \xrightarrow◆LB◆d◆RB◆ N(0, 1)$$

That is, for all $a \lt b$:

$$\lim_◆LB◆n \to \infty◆RB◆ P\left(a \lt \frac◆LB◆\bar◆LB◆X◆RB◆_n - \mu◆RB◆◆LB◆\sigma / \sqrt◆LB◆n◆RB◆◆RB◆ \lt b\right) = \Phi(b) - \Phi(a)$$

_Proof (sketch using MGFs)._ Let $Y_i = (X_i - \mu)/\sigma$So $E[Y_i] = 0$ and
$\mathrm◆LB◆Var◆RB◆(Y_i) = 1$. Define $Z_n = \frac◆LB◆1◆RB◆◆LB◆\sqrt◆LB◆n◆RB◆◆RB◆ \sum_◆LB◆i=1◆RB◆^n Y_i$. We show
$M_◆LB◆Z_n◆RB◆(t) \to e^◆LB◆t^2/2◆RB◆$ (the standard normal MGF).

Let $M(t) = E[e^◆LB◆tY_1◆RB◆]$ be the MGF of $Y_1$. Then:

$$M_◆LB◆Z_n◆RB◆(t) = \left[M\left(\frac◆LB◆t◆RB◆◆LB◆\sqrt◆LB◆n◆RB◆◆RB◆\right)\right]^n$$

By Taylor expansion of $M$ around 0:
$M(h) = 1 + h\, M'(0) + \frac◆LB◆h^2◆RB◆◆LB◆2◆RB◆ M''(0) + o(h^2) = 1 + 0 + \frac◆LB◆h^2◆RB◆◆LB◆2◆RB◆ + o(h^2)$ (since
$E[Y_1] = 0$ and $E[Y_1^2] = 1$).

Therefore:

$$M_◆LB◆Z_n◆RB◆(t) = \left[1 + \frac◆LB◆t^2◆RB◆◆LB◆2n◆RB◆ + o\left(\frac◆LB◆1◆RB◆◆LB◆n◆RB◆\right)\right]^n \to e^◆LB◆t^2/2◆RB◆$$

As $n \to \infty$. By the continuity theorem for MGFs, $Z_n \xrightarrow◆LB◆d◆RB◆ N(0, 1)$. $\blacksquare$

### 6.4 Slutsky's Theorem

**Theorem 6.4 (Slutsky).** If $X_n \xrightarrow◆LB◆d◆RB◆ X$ and $Y_n \xrightarrow◆LB◆p◆RB◆ c$ (a constant),
then:

1. $X_n + Y_n \xrightarrow◆LB◆d◆RB◆ X + c$.
2. $Y_n X_n \xrightarrow◆LB◆d◆RB◆ cX$.
3. $X_n / Y_n \xrightarrow◆LB◆d◆RB◆ X / c$ (provided $c \neq 0$).

_Intuition._ Slutsky's theorem allows us to replace a convergent-in-probability sequence by its
limit inside Expressions that converge in distribution. This is essential for deriving the
asymptotic distribution of $t$-statistics, for instance.

**Corollary 6.1 (Asymptotic distribution of the t-statistic).** If $X_1, \ldots, X_n$ are i.i.d.
With mean $\mu$ Variance $\sigma^2$And fourth moment, then

$$\frac◆LB◆\bar◆LB◆X◆RB◆_n - \mu◆RB◆◆LB◆S_n / \sqrt◆LB◆n◆RB◆◆RB◆ \xrightarrow◆LB◆d◆RB◆ N(0, 1)$$

Where $S_n^2 = \frac◆LB◆1◆RB◆◆LB◆n-1◆RB◆\sum_◆LB◆i=1◆RB◆^n (X_i - \bar◆LB◆X◆RB◆_n)^2$.

_Proof._ By the CLT, $\sqrt◆LB◆n◆RB◆(\bar◆LB◆X◆RB◆_n - \mu)/\sigma \xrightarrow◆LB◆d◆RB◆ N(0, 1)$. By the WLLN,
$S_n^2 \xrightarrow◆LB◆p◆RB◆ \sigma^2$ So $S_n \xrightarrow◆LB◆p◆RB◆ \sigma$. By the continuous mapping theorem,
$\sigma / S_n \xrightarrow◆LB◆p◆RB◆ 1$. Applying Slutsky's Theorem:

$$\frac◆LB◆\sqrt◆LB◆n◆RB◆(\bar◆LB◆X◆RB◆_n - \mu)◆RB◆◆LB◆\sigma◆RB◆ \cdot \frac◆LB◆\sigma◆RB◆◆LB◆S_n◆RB◆ \xrightarrow◆LB◆d◆RB◆ N(0, 1) \cdot 1 = N(0, 1) \quad \blacksquare$$

### 6.5 Delta Method

**Theorem 6.5 (Delta Method).** If $\sqrt◆LB◆n◆RB◆(T_n - \theta) \xrightarrow◆LB◆d◆RB◆ N(0, \sigma^2)$ and $g$
is differentiable At $\theta$ with $g'(\theta) \neq 0$Then

$$\sqrt◆LB◆n◆RB◆(g(T_n) - g(\theta)) \xrightarrow◆LB◆d◆RB◆ N(0, \sigma^2 [g'(\theta)]^2)$$

_Proof (sketch)._ By a first-order Taylor expansion:
$g(T_n) \approx g(\theta) + g'(\theta)(T_n - \theta)$. Multiplying by $\sqrt◆LB◆n◆RB◆$:
$\sqrt◆LB◆n◆RB◆(g(T_n) - g(\theta)) \approx g'(\theta) \cdot \sqrt◆LB◆n◆RB◆(T_n - \theta)$. The right side
converges in distribution to $g'(\theta) \cdot N(0, \sigma^2) = N(0, \sigma^2[g'(\theta)]^2)$.
Slutsky's theorem makes this rigorous. $\blacksquare$

## 7. Maximum Likelihood Estimation

### 7.1 Likelihood Function

Given i.i.d. Observations $x_1, \ldots, x_n$ from a distribution with parameter $\theta$The
**likelihood function** is

$$L(\theta) = \prod_◆LB◆i=1◆RB◆^n f(x_i \mid \theta)$$

And the **log-likelihood** is

$$\ell(\theta) = \log L(\theta) = \sum_◆LB◆i=1◆RB◆^n \log f(x_i \mid \theta)$$

### 7.2 MLE Procedure

The **maximum likelihood estimator** (MLE) $\hat◆LB◆\theta◆RB◆_◆LB◆\mathrm◆LB◆MLE◆RB◆◆RB◆$ maximises $L(\theta)$
(equivalently, $\ell(\theta)$):

$$\hat◆LB◆\theta◆RB◆_◆LB◆\mathrm◆LB◆MLE◆RB◆ = \arg\max_\theta L(\theta)◆RB◆$$

found by solving $\ell'(\theta) = 0$ and verifying $\ell''(\hat◆LB◆\theta◆RB◆) \lt 0$.

### 7.3 Properties of MLEs

**Theorem 7.1 (Consistency — Sketch).** Under regularity conditions,
$\hat◆LB◆\theta◆RB◆_◆LB◆\mathrm◆LB◆MLE◆RB◆ \xrightarrow◆LB◆p◆RB◆ \theta_0◆RB◆$ (the true parameter).

_Proof sketch._ By the law of large numbers,
$\frac◆LB◆1◆RB◆◆LB◆n◆RB◆\ell(\theta) \xrightarrow◆LB◆p◆RB◆ E_◆LB◆\theta_0◆RB◆[\log f(X \mid \theta)]$ For each $\theta$. The
Kullback-Leibler divergence
$D(\theta_0 \| \theta) = -E_◆LB◆\theta_0◆RB◆[\log f(X \mid \theta)] + E_◆LB◆\theta_0◆RB◆[\log f(X \mid \theta_0)]$
Is minimised (at zero) when $\theta = \theta_0$ by the information inequality. Therefore the
maximiser of $\ell(\theta)$ converges in probability to $\theta_0$.

**Theorem 7.2 (Asymptotic Normality).** Under regularity conditions:

$$\sqrt◆LB◆n◆RB◆(\hat◆LB◆\theta◆RB◆_◆LB◆\mathrm◆LB◆MLE◆RB◆ - \theta_0) \xrightarrow◆LB◆d◆RB◆ N\left(0, \frac◆LB◆1◆RB◆◆LB◆I(\theta_0)◆RB◆\right)◆RB◆$$

Where $I(\theta_0)$ is the **Fisher information**.

### 7.4 Fisher Information and the Cramer-Rao Bound

**Definition.** The **Fisher information** for a single observation is

$$I(\theta) = E\left[\left(\frac◆LB◆\partial◆RB◆◆LB◆\partial \theta◆RB◆ \log f(X \mid \theta)\right)^2\right] = -E\left[\frac◆LB◆\partial^2◆RB◆◆LB◆\partial \theta^2◆RB◆ \log f(X \mid \theta)\right]$$

Provided the interchange of differentiation and integration is valid.

**Theorem 7.3 (Cramer-Rao Lower Bound).** For any **unbiased** estimator $T$ of $\theta$:

$$\mathrm◆LB◆Var◆RB◆(T) \geq \frac◆LB◆1◆RB◆◆LB◆n\, I(\theta)◆RB◆$$

_Intuition._ The Cramer-Rao bound gives a theoretical minimum for the variance of any unbiased
estimator. An Estimator that achieves this bound is called **efficient**.

**Example 7.1.** For $X \sim N(\mu, \sigma^2)$ with $\sigma^2$ known, the Fisher information about
$\mu$ is $I(\mu) = 1/\sigma^2$. The MLE $\hat◆LB◆\mu◆RB◆ = \bar◆LB◆X◆RB◆$ has
$\mathrm◆LB◆Var◆RB◆(\bar◆LB◆X◆RB◆) = \sigma^2/n = 1/(n \cdot I(\mu))$ So the sample mean achieves the Cramer-Rao
bound and is efficient.

### 7.5 Confidence Intervals

**Definition.** A $100(1 - \alpha)\%$ **confidence interval** for $\theta$ is a random interval
$[L, U]$ such that

$$P_\theta(L \leq \theta \leq U) = 1 - \alpha$$

**Theorem 7.4.** Under the asymptotic normality of the MLE, an approximate $100(1-\alpha)\%$
confidence interval For $\theta$ is

$$\hat◆LB◆\theta◆RB◆ \pm z_◆LB◆\alpha/2◆RB◆ \cdot \frac◆LB◆1◆RB◆◆LB◆\sqrt◆LB◆n\, I(\hat◆LB◆\theta◆RB◆)◆RB◆◆RB◆$$

Where $z_◆LB◆\alpha/2◆RB◆ = \Phi^◆LB◆-1◆RB◆(1 - \alpha/2)$.

**Example 7.2.** For $X_1, \ldots, X_n \sim N(\mu, \sigma^2)$ with $\sigma$ known, the exact
$100(1-\alpha)\%$ Confidence interval for $\mu$ is:

$$\bar◆LB◆X◆RB◆ \pm z_◆LB◆\alpha/2◆RB◆ \cdot \frac◆LB◆\sigma◆RB◆◆LB◆\sqrt◆LB◆n◆RB◆◆RB◆$$

When $\sigma$ is unknown, replace $\sigma$ with $S$ and $z_◆LB◆\alpha/2◆RB◆$ with $t_◆LB◆n-1, \alpha/2◆RB◆$:

$$\bar◆LB◆X◆RB◆ \pm t_◆LB◆n-1, \alpha/2◆RB◆ \cdot \frac◆LB◆S◆RB◆◆LB◆\sqrt◆LB◆n◆RB◆◆RB◆$$

**Theorem 7.5 (Wald Confidence Interval).** For a scalar parameter $\theta$ with MLE $\hat◆LB◆\theta◆RB◆$
satisfying Asymptotic normality, the **Wald confidence interval** is

$$\hat◆LB◆\theta◆RB◆ \pm z_◆LB◆\alpha/2◆RB◆\, \widehat◆LB◆\mathrm◆LB◆SE◆RB◆(\hat◆LB◆\theta◆RB◆)◆RB◆$$

Where $\widehat◆LB◆\mathrm◆LB◆SE◆RB◆(\hat◆LB◆\theta◆RB◆) = 1/\sqrt◆LB◆n\, I(\hat◆LB◆\theta◆RB◆)◆RB◆◆RB◆$ is the estimated standard
error.

**Problem 7.4.** In a survey of 400 people, 220 support a policy. Construct a 95% confidence
interval for the True proportion $p$.

<details>
<summary>Solution</summary>

$\hat◆LB◆p◆RB◆ = 220/400 = 0.55$. For a Bernoulli: $I(p) = 1/[p(1-p)]$So
$\widehat◆LB◆\mathrm◆LB◆SE◆RB◆ = \sqrt◆LB◆\hat◆LB◆p◆RB◆(1 - \hat◆LB◆p◆RB◆)/n◆RB◆
= \sqrt◆LB◆0.55 \times 0.45 / 400◆RB◆ = \sqrt◆LB◆0.000619◆RB◆ \approx 0.02488◆RB◆$.

$$95\%\, \mathrm◆LB◆CI◆RB◆: 0.55 \pm 1.96 \times 0.02488 = 0.55 \pm 0.0488 = (0.501, 0.599)$$

</details>

## 8. Hypothesis Testing

### 8.1 Framework

A **hypothesis test** evaluates two competing statements:

- **Null hypothesis** $H_0$: the status quo (e.g., $\mu = \mu_0$).
- **Alternative hypothesis** $H_1$: what we want to show (e.g., $\mu \gt \mu_0$).

### 8.2 Test Statistics and Decisions

A **test statistic** $T$ is a function of the data. We reject $H_0$ if $T$ falls in the **rejection
Region** (critical region).

**Type I error**: rejecting $H_0$ when it is true (false positive). Probability = $\alpha$
(significance Level).

**Type II error**: failing to reject $H_0$ when it is false (false negative). Probability = $\beta$.

The **power** of a test is $1 - \beta = P(\mathrm◆LB◆reject◆RB◆  H_0 \mid H_1 \mathrm◆LB◆ is◆RB◆ true)$.

### 8.3 Neyman-Pearson Lemma

**Theorem 8.1 (Neyman-Pearson Lemma).** Consider testing $H_0: \theta = \theta_0$ versus
$H_1: \theta = \theta_1$ Based on a single observation $X$ with PDF $f(x \mid \theta)$. The **most
powerful test** of level $\alpha$ rejects $H_0$ when the **likelihood ratio** exceeds a threshold:

$$\Lambda(x) = \frac◆LB◆f(x \mid \theta_1)◆RB◆◆LB◆f(x \mid \theta_0)◆RB◆ \gt k$$

For some $k$ chosen so that $P(\Lambda(X) \gt k \mid H_0) = \alpha$.

_Proof (sketch)._ Consider any test $\phi$ with level $\alpha$ (i.e.,
$E_◆LB◆\theta_0◆RB◆[\phi(X)] \leq \alpha$). The power Under $H_1$ is
$E_◆LB◆\theta_1◆RB◆[\phi(X)] = \int \phi(x) f(x \mid \theta_1)\, dx$. Write this as
$\int \phi(x) \Lambda(x) f(x \mid \theta_0)\, dx$. The likelihood ratio test $\phi^*$ rejects when
$\Lambda \gt k$ and Randomises on the boundary, so it assigns the largest $\phi^*(x)$ values to the
largest $\Lambda(x)$ values. Any other Level-$\alpha$ test assigns less rejection probability to
large-$\Lambda$ regions and cannot exceed the power of $\phi^*$. $\blacksquare$

### 8.4 Likelihood Ratio Tests (General)

For composite hypotheses $H_0: \theta \in \Theta_0$ vs $H_1: \theta \in \Theta_1$The **generalised
likelihood Ratio statistic** is

$$\Lambda = \frac◆LB◆\sup_◆LB◆\theta \in \Theta_0◆RB◆ L(\theta)◆RB◆◆LB◆\sup_◆LB◆\theta \in \Theta◆RB◆ L(\theta)◆RB◆$$

We reject $H_0$ when $\Lambda$ is small (equivalently, when $-2\log \Lambda$ is large).

**Theorem 8.2 (Wilks' Theorem).** Under $H_0$ and regularity conditions:

$$-2 \log \Lambda \xrightarrow◆LB◆d◆RB◆ \chi^2_d$$

Where $d = \dim(\Theta) - \dim(\Theta_0)$ is the difference in the number of free parameters.

### 8.5 p-Values

The **p-value** is the probability of observing a test statistic at least as extreme as the one
Computed, assuming $H_0$ is true. We reject $H_0$ if the p-value is less than $\alpha$.

### 8.6 Z-Test for a Mean

If $X_1, \ldots, X_n \sim N(\mu, \sigma^2)$ with known $\sigma$To test $H_0: \mu = \mu_0$:

$$Z = \frac◆LB◆\bar◆LB◆X◆RB◆ - \mu_0◆RB◆◆LB◆\sigma / \sqrt◆LB◆n◆RB◆◆RB◆$$

Under $H_0$, $Z \sim N(0, 1)$.

- For $H_1: \mu \gt \mu_0$: reject if $Z \gt z_\alpha$.
- For $H_1: \mu \lt \mu_0$: reject if $Z \lt -z_\alpha$.
- For $H_1: \mu \neq \mu_0$: reject if $|Z| \gt z_◆LB◆\alpha/2◆RB◆$.

### 8.7 t-Test for a Mean (Unknown Variance)

If $\sigma$ is unknown, replace $\sigma$ with the sample standard deviation $S$:

$$T = \frac◆LB◆\bar◆LB◆X◆RB◆ - \mu_0◆RB◆◆LB◆S / \sqrt◆LB◆n◆RB◆◆RB◆$$

Under $H_0$, $T \sim t_◆LB◆n-1◆RB◆$ (Student's t-distribution with $n - 1$ degrees of freedom).

### 8.8 Chi-Squared Test for Variance

To test $H_0: \sigma^2 = \sigma_0^2$ for $X_1, \ldots, X_n \sim N(\mu, \sigma^2)$:

$$\chi^2 = \frac◆LB◆(n-1)S^2◆RB◆◆LB◆\sigma_0^2◆RB◆$$

Under $H_0$, $\chi^2 \sim \chi^2_◆LB◆n-1◆RB◆$.

### 8.9 Chi-Squared Goodness-of-Fit Test

To test whether observed data follow a specified discrete distribution, partition the sample space
into $k$ cells With expected counts $e_i$ and observed counts $o_i$. The test statistic is

$$\chi^2 = \sum_◆LB◆i=1◆RB◆^k \frac◆LB◆(o_i - e_i)^2◆RB◆◆LB◆e_i◆RB◆$$

Under $H_0$ (and provided expected counts are sufficiently large), $\chi^2 \sim \chi^2_◆LB◆k - 1 - p◆RB◆$
where $p$ is the Number of parameters estimated from the data.

**Problem 8.4.** A die is rolled 60 times with the following frequencies:
`\\{1: 8, 2: 12, 3: 9, 4: 11, 5: 13, 6: 7\\}`. Test whether the die is fair at $\alpha = 0.05$.

<details>
<summary>Solution</summary>

$H_0$: die is fair (each face has probability $1/6$). Expected count per face: $e_i = 60/6 = 10$.

$$\chi^2 = \frac◆LB◆(8 - 10)^2◆RB◆◆LB◆10◆RB◆ + \frac◆LB◆(12 - 10)^2◆RB◆◆LB◆10◆RB◆ + \frac◆LB◆(9 - 10)^2◆RB◆◆LB◆10◆RB◆ + \frac◆LB◆(11 - 10)^2◆RB◆◆LB◆10◆RB◆ + \frac◆LB◆(13 - 10)^2◆RB◆◆LB◆10◆RB◆ + \frac◆LB◆(7 - 10)^2◆RB◆◆LB◆10◆RB◆$$

$$= \frac◆LB◆4 + 4 + 1 + 1 + 9 + 9◆RB◆◆LB◆10◆RB◆ = \frac◆LB◆28◆RB◆◆LB◆10◆RB◆ = 2.8$$

Under $H_0$, $\chi^2 \sim \chi^2_5$. The critical value at $\alpha = 0.05$ is
$\chi^2_◆LB◆5, 0.05◆RB◆ = 11.07$.

Since $2.8 \lt 11.07$We fail to reject $H_0$. There is insufficient evidence that the die is unfair.
$\blacksquare$

</details>

### 8.10 Two-Sample Tests

**Two-sample Z-test.** To test $H_0: \mu_1 - \mu_2 = 0$ for independent samples with known variances
$\sigma_1^2, \sigma_2^2$:

$$Z = \frac◆LB◆\bar◆LB◆X◆RB◆_1 - \bar◆LB◆X◆RB◆_2◆RB◆◆LB◆\sqrt◆LB◆\sigma_1^2/n_1 + \sigma_2^2/n_2◆RB◆◆RB◆$$

Under $H_0$, $Z \sim N(0, 1)$.

**Two-sample t-test (Welch's).** When variances are unknown and possibly unequal:

$$T = \frac◆LB◆\bar◆LB◆X◆RB◆_1 - \bar◆LB◆X◆RB◆_2◆RB◆◆LB◆\sqrt◆LB◆S_1^2/n_1 + S_2^2/n_2◆RB◆◆RB◆$$

The degrees of freedom are approximated by Welch's formula:

$$\nu = \frac◆LB◆(S_1^2/n_1 + S_2^2/n_2)^2◆RB◆◆LB◆\frac◆LB◆(S_1^2/n_1)^2◆RB◆◆LB◆n_1 - 1◆RB◆ + \frac◆LB◆(S_2^2/n_2)^2◆RB◆◆LB◆n_2 - 1◆RB◆◆RB◆$$

**Problem 8.5.** A study compares two teaching methods. Method A (20 students): mean score 78,
standard deviation 8. Method B (25 students): mean score 72, standard deviation 10. Test
$H_0: \mu_A = \mu_B$ vs $H_1: \mu_A \neq \mu_B$ At $\alpha = 0.05$.

<details>
<summary>Solution</summary>

Using Welch's t-test:

$$\nu = \frac◆LB◆(3.2 + 4)^2◆RB◆◆LB◆3.2^2/19 + 4^2/24◆RB◆ = \frac◆LB◆51.84◆RB◆◆LB◆0.539 + 0.667◆RB◆ = \frac◆LB◆51.84◆RB◆◆LB◆1.206◆RB◆ \approx 42.98$$

**Problem 8.3.** Let $X_1, \ldots, X_n$ be i.i.d. $N(\mu, \sigma^2)$ with $\sigma^2$ known. Derive

$Y_i \sim \mathrm◆LB◆Poisson◆RB◆(\lambda/n)$ are i.i.d. $\blacksquare$

</details>

If you get this wrong, revise: Section 4.3 (MGFs) and Section 3.1 (Poisson Distribution).

## Common Pitfalls

1. Dropping negative signs during algebraic manipulation — substitute back to verify your answer.

2. Confusing the domain and range of functions, or not considering restrictions (e.g., denominator
   cannot be zero).

3. Confusing $P(A|B)$ with $P(B|A)$ — these are related by Bayes' theorem but are not equal in
   general.

## Worked Examples

### Example 1: Maximum Likelihood Estimation

**Problem.** Given i.i.d. samples $x_1, \ldots, x_n$ from $X \sim N(\mu, \sigma^2)$, find the MLE
for $\mu$ and $\sigma^2$.

**Solution.** Log-likelihood:

$$\ell(\mu, \sigma^2) = -\frac◆LB◆n◆RB◆◆LB◆2◆RB◆\ln(2\pi) - \frac◆LB◆n◆RB◆◆LB◆2◆RB◆\ln\sigma^2 - \frac◆LB◆1◆RB◆◆LB◆2\sigma^2◆RB◆\sum_◆LB◆i=1◆RB◆^n (x_i - \mu)^2$$

$\frac◆LB◆\partial \ell◆RB◆◆LB◆\partial \mu◆RB◆ = \frac◆LB◆1◆RB◆◆LB◆\sigma^2◆RB◆\sum(x_i - \mu) = 0 \Rightarrow \hat◆LB◆\mu◆RB◆ = \bar◆LB◆x◆RB◆$.

$\frac◆LB◆\partial \ell◆RB◆◆LB◆\partial \sigma^2◆RB◆ = -\frac◆LB◆n◆RB◆◆LB◆2\sigma^2◆RB◆ + \frac◆LB◆1◆RB◆◆LB◆2\sigma^4◆RB◆\sum(x_i - \mu)^2 = 0 \Rightarrow \hat◆LB◆\sigma◆RB◆^2 = \frac◆LB◆1◆RB◆◆LB◆n◆RB◆\sum(x_i - \bar◆LB◆x◆RB◆)^2$.

Note: $\hat◆LB◆\sigma◆RB◆^2$ is biased (divide by $n$); unbiased version divides by $n-1$.

$\blacksquare$

### Example 2: Central Limit Theorem Application

**Problem.** A fair die is rolled 100 times. Use the CLT to approximate $P(\bar◆LB◆X◆RB◆ > 3.7)$.

**Solution.** For a fair die: $\mu = 3.5$, $\sigma^2 = \frac◆LB◆35◆RB◆◆LB◆12◆RB◆ = 2.917$, $\sigma = 1.708$.

By CLT: $\bar◆LB◆X◆RB◆ \approx N(\mu, \sigma^2/n)$, so $\text◆LB◆SE◆RB◆ = 0.1708$.

$$P(\bar◆LB◆X◆RB◆ > 3.7) = P\left(Z > \frac◆LB◆3.7 - 3.5◆RB◆◆LB◆0.1708◆RB◆\right) = P(Z > 1.17) = 1 - \Phi(1.17) = 0.121$$

$\blacksquare$

## Summary

- Probability axioms: non-negativity, normalisation, additivity for disjoint events; conditional
  probability and Bayes' theorem.
- Distributions: discrete (Binomial, Poisson, Geometric) and continuous (Normal, Exponential,
  Gamma); know mean, variance, and MGF.
- Estimation: MLE maximises likelihood function; method of moments equates sample and population
  moments.
- Hypothesis testing: null/alternative hypotheses, test statistic, p-value, significance level; Type
  I/II errors and power.
- Central Limit Theorem: sample mean of i.i.d. variables is approximately normal for large $n$,
  regardless of the underlying distribution.
