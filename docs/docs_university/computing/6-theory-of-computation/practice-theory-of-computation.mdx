<PracticeProblem
  question="To prove that L = {a^n b^n : n >= 0} is not regular using the pumping lemma, for any pumping length p, which string s should you choose?"
  options={['a^p b^p', 'a^(p+1) b^p', 'a^p b^(p+1)', '(ab)^p']}
  correctAnswer={0}
  explanation="Choose s = a^p b^p with |s| = 2p >= p. By the pumping lemma, s = xyz with |xy| <= p and |y| > 0. Since the first p symbols are all a's, y consists entirely of a's: y = a^k for some k > 0. Pumping down (i=0) gives xy^0z = a^(p-k) b^p, which has fewer a's than b's and is not in L. Alternatively, pumping up gives a^(p+k) b^p, also not in L. This contradicts the pumping lemma, proving L is not regular. The key insight is choosing a string where |xy| <= p forces y to lie entirely within the a's."
  difficulty="medium"
