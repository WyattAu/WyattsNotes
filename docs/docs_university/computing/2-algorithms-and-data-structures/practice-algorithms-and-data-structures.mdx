<PracticeProblem
  question="Using the Master theorem, what is the solution to T(n) = 4T(n/2) + n^2?"
  options={['O(n^2 log n)', 'O(n^2)', 'O(n^3)', 'O(n log n)']}
  correctAnswer={0}
  explanation="T(n) = aT(n/b) + f(n) with a=4, b=2, f(n)=n^2. Compute n^(log_b(a)) = n^(log_2(4)) = n^2. Since f(n) = n^2 = Theta(n^2) = Theta(n^(log_b(a)), we are in Case 2 of the Master theorem. Case 2 gives T(n) = Theta(n^(log_b(a)) * log n) = Theta(n^2 log n), so T(n) = O(n^2 log n)."
  difficulty="medium"
/>

### Dynamic Programming and Graph Algorithms

<PracticeProblem
