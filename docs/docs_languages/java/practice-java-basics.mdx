### Garbage Collection

<PracticeProblem

  options={[
    'Eden space in the young generation',
    'Old generation (tenured space)',
    'Survivor space S0',
    'Stack memory',
  ]}
  correctAnswer={0}
  explanation="All new objects (except very large ones that bypass the young generation) are allocated in Eden, the first region of the young generation. When Eden fills up, a minor GC traces reachable objects and copies survivors to a survivor space. After surviving a configurable number of minor GCs (controlled by `-XX:MaxTenuringThreshold`), objects are promoted to the old (tenured) generation."
