### Functions and Modules

<PracticeProblem question="What is the difference between `def` and `defp` in a module?" options={[ 'def is public, defp is private to the defining module', 'def is for functions, defp is for protocols', 'def compiles at runtime, defp compiles at compile time', 'def takes any number of args, defp takes exactly one', ]} correctAnswer={0} explanation="def defines a public function callable from any module. defp defines a private function only callable from within the defining module.
