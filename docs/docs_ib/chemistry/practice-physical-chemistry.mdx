### Equilibrium

<PracticeProblem
  question="For the reaction N₂(g) + 3H₂(g) ⇌ 2NH₃(g), at equilibrium the concentrations are [N₂] = 1.0 M, [H₂] = 1.0 M, [NH₃] = 0.50 M. Calculate Kc."
  options={['0.25', '0.50', '1.0', '2.0']}
  correctAnswer={0}
  explanation="Kc = [NH₃]²/([N₂][H₂]³) = (0.50)²/(1.0 × 1.0³) = 0.25/1.0 = 0.25."
  difficulty="medium"
/>

<PracticeProblem
  question="For the exothermic equilibrium: 2SO₂(g) + O₂(g) ⇌ 2SO₃(g), what effect does increasing temperature have?"

  correctAnswer={0}
