<PracticeProblem
  question="A student measures the absorbance of five standard solutions and plots a calibration curve. The equation of the line of best fit is A = 0.050c + 0.002, where A is absorbance and c is concentration in mol dm⁻³. An unknown solution has an absorbance of 0.352. What is its concentration?"
  options={['7.00 mol dm⁻³', '7.06 mol dm⁻³', '0.019 mol dm⁻³', '0.018 mol dm⁻³']}
  correctAnswer={0}
  explanation="Substitute A = 0.352 into the equation: 0.352 = 0.050c + 0.002. Rearranging: 0.050c = 0.352 - 0.002 = 0.350. c = 0.350/0.050 = 7.00 mol dm⁻³. Note that the small positive intercept of 0.002 represents the blank correction (absorbance at zero concentration).
