## 概述

本笔记创建于IB文凭课程学习期间，与IBO官方教学大纲对齐。涵盖计算机科学（HL/SL）、数学（AA/AI）及其他考试科目，包含推导过程、计算示例和考试模拟练习题。

## IB评估结构

IB各科目通过外部考试和内部评估（IA）相结合的方式进行评估：

- **外部考试** — 每科2-3份试卷，在五月或十一月参加
- **内部评估（IA）** — 个人探索或项目，由学校评分，IB组织 moderates
- **总分** — 每科评分1-7分，文凭总分24-45分（加上知识和认识论及拓展论文奖励分）

## 如何使用这些笔记

1. 从侧边栏选择你的科目。
2. 每页内容按照IB教学大纲顺序排列，并标注主题编号。
3. 定义精确且适合考试使用。例题采用IB标准记法。
4. 请与对应年份的官方IB科目指南对照参考。

## 学习策略

1. **从教学大纲开始** — 阅读每门科目的官方IB科目指南。其中列出了所有可考核的学习目标。
2. **使用指令术语** — IB考试题目使用特定的指令术语（如"定义"、"解释"、"评估"、"讨论"）。每个术语都有精确的含义，决定了所需回答的深度。
3. **练习历年真题** — IB会发布历年考试试卷（第1、2、3卷）及评分方案。请在限时条件下使用它们进行练习。
4. **尽早规划IA** — 内部评估是重要的评分组成部分（占最终成绩的20-25%）。选择一个你真正感兴趣的主题，并尽早开始。
5. **制定复习时间表** — IB各科目同时进行考试。平衡你的复习安排，避免忽视任何一科。

## IB评分

每门IB科目按**1到7分**的等级评分：

| 等级 | 描述           |
| ---- | -------------- |
| 7    | 优秀表现       |
| 6    | 良好表现       |
| 5    | 较好表现       |
| 4    | 满意表现       |
| 3    | 一般表现       |
| 2    | 较差表现       |
| 1    | 非常差的表现   |

文凭要求最低**24分**（来自六门科目），加上知识和认识论（TOK）及拓展论文（EE）的奖励分。最高分为**45分**（42 + 3奖励分）。
